# Contributing to Yumil MPM

Thank you for your interest in Yumil MPM.

ご関心ありがとうございます。

---

## Repository structure / リポジトリ構成

This public repository hosts:

- `README.md` / `README_ja.md` — overview and download links
- `LICENSE` — End-User License Agreement
- `PRIVACY_POLICY.md` / `PRIVACY_POLICY_ja.md` — privacy policy
- `assets/` — icon and screenshots
- `instructions/` — in-app help content (added at Launch)

The application source code is maintained in a **private repository** and is not part of this public repo.

このリポジトリは README・ライセンス・プライバシーポリシー・アセットを含みます。**アプリ本体のソースコードは Private リポで管理**しており、ここには含まれません。

---

## Pull Requests we accept / 受け付ける Pull Request

✅ **Accepted** / **受付中**:

- Typo fixes in `README.md` / `README_ja.md`
- Improvements to `PRIVACY_POLICY.md` wording for clarity
- Translation improvements for existing content

❌ **Not accepted here** / **このリポでは不受理**:

- Application code changes (the source is upstream in a private repo)
- New screenshots or asset replacements (these are produced as part of release)
- Architectural or feature design proposals (please open an Issue or Discussion instead)

---

## Pull Requests for the extensions / 拡張機能リポへの PR

The two extension repositories are licensed under MIT and welcome code contributions:

- [`comfyui-yumil-mpm`](https://github.com/maigonia/comfyui-yumil-mpm)
- [`sd-webui-yumil-mpm`](https://github.com/maigonia/sd-webui-yumil-mpm)

See each repository's `CONTRIBUTING.md` for guidelines.

拡張機能 2 リポは MIT ライセンスで、コード貢献を歓迎しています。それぞれの `CONTRIBUTING.md` を参照してください。
