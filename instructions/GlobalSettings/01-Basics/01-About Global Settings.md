# About Global Settings

Global Settings is a menu that consolidates settings that apply to the entire application.

Basically, each top menu has its own settings menus, but menus that affect the entire app are managed as Global Settings.

## Types of Settings

| Setting               | Description                                            |
| --------------------- | ------------------------------------------------------ |
| Language              | Switch UI display language (Japanese / English)        |
| Font Settings         | Change fonts and sizes for each panel                  |
| EditPlugin Settings   | Manage enabled/disabled state of text editing plugins  |
| LM Studio Settings    | Configure integration with local AI (LM Studio)       |
| URL Whitelist Manager | Manage allowed domains for opening external URLs       |
| Tag Manager           | Manage tags across the entire project                  |
| WD14 Tagger Settings  | Adjust thresholds for automatic image tag extraction   |
| Sandbox Settings      | Set memory limit for Programmable Block execution      |
| API Key Settings      | Securely manage API keys for external services         |

## Accessing the Menu

Click **GlobalSettings** in the menu bar to display the menu. Select each item to open a dialog where you can change settings.

## Saving Settings

Most settings are saved when you click "OK" or "Save" in the dialog. Only language switching takes effect immediately upon menu selection.
