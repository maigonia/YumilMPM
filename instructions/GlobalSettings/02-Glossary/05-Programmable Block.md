# Programmable Block

## What Is It
Programmable Block is a feature that allows users to write and execute custom JavaScript code. You can dynamically process prompt content using custom logic.

## Uses
- Perform conditional branching and random selection during prompt generation
- Automate text processing and transformation
- Build dynamic prompts that reference external data

## Relationship with [Sandbox](03-Sandbox.md)
Programmable Block scripts are executed inside a [Sandbox](03-Sandbox.md) for security. Scripts that exceed the memory limit are interrupted.
