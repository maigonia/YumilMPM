# Japanese
**Language > Japanese**

## When to Use
- When you want to switch the UI display language to Japanese

## What It Does
Changes the entire app's display language to Japanese. All UI elements including menus, buttons, and dialogs will be displayed in Japanese.

## How to Use
1. Select GlobalSettings > Language > **Japanese**
2. The entire UI immediately switches to Japanese

## Notes
- No dialog is opened; the change takes effect the moment you select it
- The setting is automatically saved and persists on next launch
- A checkmark is displayed next to the currently selected language
