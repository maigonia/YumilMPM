# English
**Language > English**

## When to Use
- When you want to switch the UI display language to English

## What It Does
Changes the entire app's display language to English. All UI elements including menus, buttons, and dialogs will be displayed in English.

## How to Use
1. Select GlobalSettings > Language > **English**
2. The entire UI immediately switches to English

## Notes
- No dialog is opened; the change takes effect the moment you select it
- The setting is automatically saved and persists on next launch
- A checkmark is displayed next to the currently selected language
