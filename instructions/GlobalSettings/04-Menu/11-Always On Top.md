# Always On Top
**Always On Top**

## When to Use
- When you want to keep this app always visible while using other apps
- When you want to work side by side with an image viewer or reference materials

## What It Does
Keeps the app window always on top of all other windows. A checkmark in the menu indicates it's enabled; no checkmark means disabled.

## How to Use
Click GlobalSettings > **Always On Top** to toggle ON/OFF.

You can also toggle it from the command palette with `settings: Always On Top`.

## Notes
- The setting is saved to the project and persists when reopening the project
- This operation is available even in compact mode
- The same function can be toggled from the pin icon at the top of the editor in compact mode
