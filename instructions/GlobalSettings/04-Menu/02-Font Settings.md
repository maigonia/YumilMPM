# Font Settings
**Font Settings**

## When to Use
- When you want to change the font size of editors or trees
- When you want to switch to a more readable font
- When text is too small on a high-resolution monitor

## What It Does
Allows you to individually set the font type and size for each panel.

Configurable panels:

| Panel | Description | Default |
|-------|-------------|---------|
| Panel Headers | Panel headers | system-ui / 14px |
| Category Tree | Category tree | system-ui / 13px |
| Prompt Tree | Prompt tree | system-ui / 13px |
| Prompt Browser | Prompt browser | system-ui / 13px |
| Prompt Editor | Prompt editor | Consolas, monospace / 14px |
| Generation | Generation controller | system-ui / 13px |
| Advanced Panel | Advanced panel | system-ui / 13px |

## How to Use
1. Select GlobalSettings > **Font Settings** to open the dialog
2. Each row in the table corresponds to a panel
3. Select a font from the **Font** column dropdown (system-installed fonts are listed)
4. Enter the font size in the **Size** column (range: 8-32px)
5. Click "OK" to save

## Notes
- Changes are not applied until you click "OK"
- "Reset to Defaults" restores all panels to default font settings
- In the font dropdown, each font name is displayed in its own typeface so you can preview the appearance
