# Recommended Initial Settings

Global Settings contains many settings that affect the entire app, so reviewing a few settings before you start will make things smoother.

## Recommended Order

### 1. Check the Language
Check the display language at GlobalSettings > **Language**. Japanese and English are supported.

### 2. Adjust Fonts
Adjust fonts and sizes at GlobalSettings > **Font Settings**.

The **PromptEditor** font in particular is something you'll constantly be looking at, so setting a comfortable font and size is recommended.

### 3. Enable External Links
By default, the ability to open external URLs from the app is disabled. If you use links to Civitai or Hugging Face, enable them at GlobalSettings > **URL Whitelist Manager**.

### 4. Configure LM Studio (If Using AI Integration)
If you want to use AI features, first install and launch LM Studio, then configure presets and templates at GlobalSettings > **LM Studio Settings**.

## Settings to Configure Later
- **Tag Manager** — Use when tags accumulate and need organizing
- **WD14 Tagger Settings** — Only if using image tag extraction
- **Sandbox Settings** — The default 512MB is usually sufficient
- **API Key Settings** — Only if external service API keys are needed
