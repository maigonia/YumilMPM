# Go to Checked Below
**Go to Checked Below**

## When to Use
- When you want to review checked prompts in order
- When you want to quickly move to the next checked prompt

## What It Does
Moves to the nearest checked prompt below the current selection.

## How to Use
1. Select **PromptTree > Goto Checked Below** from the menu
2. Or click the down arrow button in the toolbar

## Notes
- Not available when there are no checked prompts
