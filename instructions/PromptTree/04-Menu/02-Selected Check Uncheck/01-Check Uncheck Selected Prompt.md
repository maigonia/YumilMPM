# Selected Prompt
**Selected Prompt**

## When to Use
- When you want to toggle the check state of the selected prompt even when the prompt tree doesn't have focus
- When you want to perform check operations from a menu or keyboard shortcut

## What It Does
Toggles the check state of the currently selected prompt itself. Checked items are unchecked, and unchecked items are checked.

## How to Use
1. Select a prompt
2. Select **PromptTree > Selected: Check/Uncheck > Selected Prompt** from the menu

## Notes
- When the prompt tree has focus, you can also use the **Space** key for the same operation
- This menu item works regardless of focus location, so it can be used even when focus is on the editor or elsewhere
