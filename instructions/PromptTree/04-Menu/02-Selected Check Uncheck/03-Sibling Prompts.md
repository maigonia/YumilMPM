# Sibling Prompts
**Sibling Prompts**

## When to Use
- When you want to batch check/uncheck prompts at the same level as the selected prompt
- When you want to target all prompts with the same parent

## What It Does
Toggles the check state of all prompts at the same level (with the same parent) as the selected prompt.

## How to Use
1. Select a prompt
2. Select **PromptTree > Selected: Check/Uncheck > Sibling Prompts** from the menu
3. Or click the people icon in the toolbar

## Notes
- The selected prompt itself is also included
- At the root level, all root prompts are targeted
