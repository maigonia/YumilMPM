# Leaf Prompts
**Leaf Prompts**

## When to Use
- When you want to check/uncheck only leaf (childless) prompts under the selected prompt
- When you want to target leaf prompts rather than prompts with children

## What It Does
Toggles the check state of only leaf nodes (prompts without children) under the selected prompt.

## How to Use
1. Select a prompt
2. Select **PromptTree > Selected: Check/Uncheck > Leaf Prompts** from the menu
3. Or click the leaf icon in the toolbar

## Notes
- Prompts with children are excluded
- If the selected prompt itself is a leaf, it becomes the target
