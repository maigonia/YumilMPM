# Child Prompts
**Child Prompts**

## When to Use
- When you want to batch check/uncheck the direct children of the selected prompt
- When you want to target only the direct children of a specific level

## What It Does
Toggles the check state of the direct children (one level below only) of the selected prompt. Checked items are unchecked, and unchecked items are checked.

## How to Use
1. Select a prompt that has children
2. Select **PromptTree > Selected: Check/Uncheck > Child Prompts** from the menu
3. Or click the baby icon in the toolbar

## Notes
- Only direct children are targeted (grandchildren and below are not included)
- Has no effect on prompts without children
