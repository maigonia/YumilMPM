# Descendants Prompts
**Descendants Prompts**

## When to Use
- When you want to batch check/uncheck all prompts under the selected prompt
- When you want to target an entire prompt and its contents

## What It Does
Toggles the check state of all descendants (children, grandchildren, great-grandchildren, etc.) of the selected prompt.

## How to Use
1. Select a prompt
2. Select **PromptTree > Selected: Check/Uncheck > Descendants Prompts** from the menu
3. Or click the tree icon in the toolbar

## Notes
- The selected prompt itself is not included
- All hierarchy levels are targeted
