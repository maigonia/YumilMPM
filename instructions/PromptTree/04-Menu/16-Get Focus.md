# Get Focus
**Get Focus**

## When to Use
- When you want to operate the prompt tree with the keyboard
- When you want to navigate without using the mouse

## What It Does
Moves keyboard focus to the prompt tree. After focusing, you can navigate using arrow keys.

## How to Use
1. Select **PromptTree > Get Focus** from the menu
2. Or press **Alt+2**
