# Go to Checked Above
**Go to Checked Above**

## When to Use
- When you want to review checked prompts in order
- When you want to quickly move to the previous checked prompt

## What It Does
Moves to the nearest checked prompt above the current selection.

## How to Use
1. Select **PromptTree > Goto Checked Above** from the menu
2. Or click the up arrow button in the toolbar

## Notes
- Not available when there are no checked prompts
