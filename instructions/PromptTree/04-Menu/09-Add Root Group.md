# Add Root Group

**Add to root**

## When to Use

- When you want to add a new theme or genre prompt at the top level
- When you want to place a prompt at the very top of the hierarchy

## What It Does

Adds an empty prompt at the top level (root level) of the category.

## How to Use

1. Select any prompt (or nothing selected is fine too)
2. Select **PromptTree > Add Root Group** from the menu
3. A prompt is added directly under the category

## Notes

- Regardless of which prompt is selected, it is always added at the root level
