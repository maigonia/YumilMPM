# Delete Prompts
**Delete Prompts**

## When to Use
- When you want to delete an unnecessary prompt
- When you want to delete a prompt for organization

## What It Does
Deletes the selected prompt. If the prompt has descendants, all descendants are also deleted.

## How to Use
1. Select the prompt to delete
2. Select **PromptTree > Selected > Delete Prompts** from the menu (or the **Delete** key)
3. Click **Delete** in the confirmation dialog

## Notes
- Deletion cannot be undone
- When deleting a prompt with descendants, the number of descendants is displayed
- For [Folder Prompt](../../01-Basics/01-Concepts/03-What%20Is%20a%20Prompt.md)s and [File Prompt](../../01-Basics/01-Concepts/03-What%20Is%20a%20Prompt.md)s, only the prompt is deleted and the original files remain (use "Delete Chunk and Files" if you also want to delete the files)
