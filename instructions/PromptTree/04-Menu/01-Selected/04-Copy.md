# Copy
**Copy**

## When to Use
- When you want to duplicate a prompt for use elsewhere
- When you want to use it as a basis for creating a similar prompt

## What It Does
Copies the selected prompt to the clipboard. If it has descendants, they are included in the copy.

## How to Use
1. Select the prompt to copy
2. Select **PromptTree > Selected > Copy** from the menu (or **Ctrl+C**)

## Notes
- After copying, use "Paste" to duplicate at any location
- Copied prompts are assigned new IDs
- [Folder Prompt](../../01-Basics/01-Concepts/03-What%20Is%20a%20Prompt.md)s become [Group Prompt](../../01-Basics/01-Concepts/03-What%20Is%20a%20Prompt.md)s, and [File Prompt](../../01-Basics/01-Concepts/03-What%20Is%20a%20Prompt.md)s become [Normal Prompt](../../01-Basics/01-Concepts/03-What%20Is%20a%20Prompt.md)s
- Tags, memos, images, and other metadata are preserved
