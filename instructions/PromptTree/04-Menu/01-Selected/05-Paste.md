# Paste

**Paste**

## When to Use

- When you want to place a copied or cut prompt at a different location
- When you want to move or copy a prompt to a different category

## What It Does

Pastes prompts from the clipboard at the selected location.

## How to Use

1. Select the parent prompt for the paste destination (if nothing is selected, it will be pasted at the root)
2. Select **PromptTree > Selected > Paste** from the menu (or **Ctrl+V**)
3. The prompts are pasted as children of the selected prompt

## Notes

- Not available when the clipboard has no prompts
- If a prompt with the same name exists, it is automatically renamed
- Can paste from other categories as well
