# Select Parent Prompt
**Select Parent Prompt**

## When to Use
- When you want to quickly go back from a child prompt to its parent
- When you want to navigate up the hierarchy

## What It Does
Selects the parent prompt of the currently selected prompt.

## How to Use
1. Select **PromptTree > Selected > Select Parent Prompt** from the menu

## Notes
- Not available for root-level prompts (those without a parent)
- The parent prompt is selected and its content is displayed in the editor
