# Select Previous Prompt
**Select Previous**

## When to Use
- When you want to navigate between prompts without using the keyboard
- When you want to move to the previous prompt from the menu

## What It Does
Selects the prompt immediately before the currently selected prompt in the tree.

## How to Use
1. Select **PromptTree > Selected > Select Previous Prompt** from the menu

## Notes
- The **↑** key can also be used for the same operation
- Nothing happens if already at the top
