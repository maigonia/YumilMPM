# Delete Chunk and Files
**Delete Chunk and Files**

## When to Use
- When you want to delete a [Folder Prompt](../../01-Basics/01-Concepts/03-What%20Is%20a%20Prompt.md) or [File Prompt](../../01-Basics/01-Concepts/03-What%20Is%20a%20Prompt.md) along with its original files
- When you want to delete a prompt together with associated files (images, memos)

## What It Does
Deletes a prompt together with its associated files. You can individually select which files to delete in the dialog.

Files available for deletion:
- Base file / source file
- Associated memo files
- Associated image files

## How to Use
1. Select the prompt to delete
2. Select **PromptTree > Selected > Delete Chunk and Files...** from the menu
3. Check the files you want to delete in the dialog
4. Click **Delete**

## Notes
- This menu is only available for prompts that have file paths
- Not available for regular prompts (manually created ones)
- File deletion cannot be undone
- You can also open the base file folder from the dialog
