# Move Up
**Move Up**

## When to Use
- When you want to change the display order of prompts
- When you want to move an important prompt higher up

## What It Does
Moves the selected prompt one position up within the same hierarchy level.

## How to Use
1. Select the prompt to move
2. Select **PromptTree > Selected > Move Up** from the menu

## Notes
- Not available if already at the top
- Cannot move across hierarchy levels (use "Cut" then "Paste" for moving to a different parent)
