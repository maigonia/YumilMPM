# Set as Normal Prompt
**Set as Normal Prompt**

## When to Use
- When you want to revert a prompt set as a group back to a normal prompt
- When you want to change the icon for organization

## What It Does
Sets the selected prompt as a normal prompt (file icon).

## How to Use
1. Select the prompt whose icon you want to change
2. Select **PromptTree > Selected > Change Icon > Set as Normal Prompt** from the menu

## Notes
- Cannot be used on [Folder Prompt](../../../01-Basics/01-Concepts/03-What Is a Prompt.md)s or [File Prompt](../../../01-Basics/01-Concepts/03-What Is a Prompt.md)s
- Child prompts are preserved even after changing the icon
