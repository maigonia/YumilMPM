# Cut
**Cut**

## When to Use
- When you want to move a prompt to a different location
- When you want to change the hierarchical structure of prompts

## What It Does
Cuts the selected prompt and saves it to the clipboard. The original prompt is deleted after pasting.

## How to Use
1. Select the prompt to cut
2. Select **PromptTree > Selected > Cut** from the menu (or **Ctrl+X**)
3. Select the destination and perform "Paste"

## Notes
- Unlike Copy, file reference information is preserved
- The original prompt remains until Paste is executed
- If there are descendants, they are moved together
