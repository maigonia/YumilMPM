# Rename
**Rename**

## When to Use
- When you want to change a prompt's name to something more descriptive
- When you want to standardize names for organization

## What It Does
Changes the name of the selected prompt. A dialog appears for entering the new name.

## How to Use
1. Select the prompt to rename
2. Select **PromptTree > Selected > Rename** from the menu (or the **F2** key)
3. Enter the new name in the dialog
4. Click **OK** (or press **Enter**)

## Notes
- An error occurs if a prompt with the same name exists at the same hierarchy level
- Empty names or names containing invalid characters cannot be set
- Press **Esc** to cancel
