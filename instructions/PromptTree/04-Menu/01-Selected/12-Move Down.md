# Move Down
**Move Down**

## When to Use
- When you want to change the display order of prompts
- When you want to move a prompt further down

## What It Does
Moves the selected prompt one position down within the same hierarchy level.

## How to Use
1. Select the prompt to move
2. Select **PromptTree > Selected > Move Down** from the menu

## Notes
- Not available if already at the bottom
- Cannot move across hierarchy levels (use "Cut" then "Paste" for moving to a different parent)
