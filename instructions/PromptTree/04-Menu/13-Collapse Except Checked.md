# Collapse Except Checked
**Collapse All Except Checked**

## When to Use
- When you want to see only checked prompts
- When you want to tidy up the tree and review the check state

## What It Does
Collapses everything except the paths leading to checked prompts.

## How to Use
1. Select **PromptTree > Collapse Except Checked** from the menu

## Notes
- Parents of checked prompts remain expanded
