# Content
**Content**

## When to Use
- When you want to use the prompt's content elsewhere
- When you want to copy the content as text

## What It Does
Copies the content (text) of the selected prompt to the clipboard.

## How to Use
1. Select a prompt
2. Select **PromptTree > Selected: Copy Info > Content** from the menu
