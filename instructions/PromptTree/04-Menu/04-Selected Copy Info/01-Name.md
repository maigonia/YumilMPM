# Name
**Name**

## When to Use
- When you want to use the prompt's name elsewhere
- When you want to copy the name as text

## What It Does
Copies the name of the selected prompt to the clipboard.

## How to Use
1. Select a prompt
2. Select **PromptTree > Selected: Copy Info > Name** from the menu
