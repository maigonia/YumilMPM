# Check Filtered Results
**Check Filtered**

## When to Use
- When you want to check only the prompts that matched the search
- When you want to batch check prompts matching specific criteria

## What It Does
Checks all prompts displayed in the current search results.

## How to Use
1. Execute a search in the search box
2. Select **PromptTree > Search Panel > Check Filtered Results** from the menu
3. Or click the "✓ Filtered" button in the toolbar

## Notes
- Has no effect when there are no search results
