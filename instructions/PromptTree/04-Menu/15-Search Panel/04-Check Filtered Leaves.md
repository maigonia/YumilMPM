# Check Filtered Leaves
**Check Filtered Leaves**

## When to Use
- When you want to check only leaf prompts from the search results
- When you want to exclude prompts with children and check only leaf prompts

## What It Does
Checks the leaf prompts (prompts without children) displayed in the current search results.

## How to Use
1. Execute a search in the search box
2. Select **PromptTree > Search Panel > Check Filtered Leaves** from the menu
3. Or click the "✓ Filtered Leaves" button in the toolbar

## Notes
- Prompts with children are not checked
- Has no effect when there are no search results
