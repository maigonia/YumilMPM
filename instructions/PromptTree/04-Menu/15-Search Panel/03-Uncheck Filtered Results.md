# Uncheck Filtered Results
**Uncheck Filtered**

## When to Use
- When you want to batch uncheck the prompts that matched the search
- When you want to exclude prompts matching specific criteria

## What It Does
Unchecks all prompts displayed in the current search results.

## How to Use
1. Execute a search in the search box
2. Select **PromptTree > Search Panel > Uncheck Filtered Results** from the menu

## Notes
- Has no effect when there are no search results
