# Get Focus (Search Panel)
**Get Focus**

## When to Use
- When you want to quickly start searching with the keyboard

## What It Does
Moves focus to the search box in the search panel.

## How to Use
1. Select **PromptTree > Search Panel > Get Focus** from the menu
2. Or press **Ctrl+F**
