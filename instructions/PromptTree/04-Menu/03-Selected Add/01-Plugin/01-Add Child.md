# Add Child

**Add child prompt**

## When to Use

- When you want to add variations or details under an existing prompt
- When you want to organize prompts in a hierarchical structure

## What It Does

Adds one empty child prompt under the selected prompt.

## How to Use

1. Select the parent prompt
2. Select **PromptTree > Selected: Add > Add Child** from the menu
3. A new child prompt "new item" is added
4. Rename as needed

## Notes

- If a prompt with the same name already exists, a number like "new item (2)" is appended
- After adding, the parent prompt is automatically expanded
