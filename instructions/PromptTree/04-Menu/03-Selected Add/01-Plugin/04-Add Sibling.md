# Add Sibling

**Add sibling prompt**

## When to Use

- When you want to add more parallel choices at the same level
- When you want to add prompts of a similar category at the same level

## What It Does

Adds an empty prompt at the same level (sibling) as the selected prompt.

## How to Use

1. Select a prompt
2. Select **PromptTree > Selected: Add > Add Sibling** from the menu
3. A prompt is added at the same level below the selected prompt

## Notes

- It is added as a child of the parent (at the same level as the selected prompt)
