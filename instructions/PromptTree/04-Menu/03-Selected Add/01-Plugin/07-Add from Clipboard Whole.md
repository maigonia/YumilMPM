# Add from Clipboard (Whole)

**Add entire clipboard as one child prompt**

## When to Use

- When you want to import long text or an entire prompt as a single prompt
- When you want to save multi-line text as a whole without splitting it

## What It Does

Adds the entire clipboard text as a single child prompt.

## How to Use

1. Copy text to the clipboard
2. Select the parent prompt
3. Select **PromptTree > Selected: Add > Add from Clipboard (Whole)** from the menu
4. The entire clipboard content is added as one child prompt

## Notes

- Even with multiple lines, it becomes one prompt
- The prompt name is auto-generated from the beginning of the content
