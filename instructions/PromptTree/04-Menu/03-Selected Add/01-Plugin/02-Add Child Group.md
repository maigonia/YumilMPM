# Add Child Group

**Add group**

## When to Use

- When you want to classify and organize prompts like folders
- When you want to create a group to bundle related prompts

## What It Does

Adds a group (folder icon) child prompt under the selected prompt.

## How to Use

1. Select the parent prompt
2. Select **PromptTree > Selected: Add > Add Child Group** from the menu
3. A child prompt with a group icon is added

## What Is a Group

A group is the same as a regular prompt but displayed with a folder icon. It serves as a visual marker for organization.
