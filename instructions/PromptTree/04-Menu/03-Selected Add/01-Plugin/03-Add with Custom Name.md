# Add with Custom Name

**Add sibling prompt with specified name**

## When to Use

- When you want to decide the name before creating a prompt
- When you want to give a meaningful name from the start instead of "new item"

## What It Does

Displays a dialog to enter a name, then adds a sibling prompt at the same level as the selected prompt with the specified name.

## How to Use

1. Select the reference prompt
2. Select **PromptTree > Selected: Add > Add with Custom Name** from the menu
3. A name input dialog appears
4. Enter the name and click **OK**
5. A sibling prompt with the entered name is added

## Notes

- Cancelling adds nothing
- If a prompt with the same name already exists, a number is appended
