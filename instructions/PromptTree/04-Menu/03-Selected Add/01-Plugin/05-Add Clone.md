# Add Clone

**Duplicate as sibling prompt**

## When to Use

- When you want to create a variation based on an existing prompt with slight changes
- When you want to duplicate a prompt with the same content for a different purpose

## What It Does

Copies the selected prompt and creates a duplicate at the same level (sibling).

## How to Use

1. Select the prompt to duplicate
2. Select **PromptTree > Selected: Add > Add Clone** from the menu
3. A duplicate of the selected prompt is added at the same level

## Notes

- The name will have a number appended like "original name (2)"
- Content, tags, memos, and everything else are copied
- Child prompts are not duplicated (only the selected prompt itself)
