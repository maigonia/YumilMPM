# Add from Text File (Lines)

**Add child prompts line by line from a text file**

## When to Use

- When you want to batch import a tag list or word list text file
- When you want to make each line of a file a separate prompt

## What It Does

Selects a text file and adds each line as a child prompt.

## How to Use

1. Select the parent prompt
2. Select **PromptTree > Selected: Add > Add from Text File (Lines)** from the menu
3. A file selection dialog appears
4. Select a text file (.txt)
5. Each line is added as an individual child prompt

## Notes

- Empty lines are ignored
- A large number of prompts will be created if there are many lines
