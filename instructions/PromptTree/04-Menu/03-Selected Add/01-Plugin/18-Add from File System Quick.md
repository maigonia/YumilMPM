# Add from File System (Quick)

**Instantly import from file system using previous settings**

## When to Use

- When you want to repeatedly import with the same folder and same settings
- When you want to skip the settings dialog and immediately execute an import

## What It Does

Immediately executes import using the same settings from the last **Add from File System** operation.

## How to Use

1. Select the parent prompt
2. Select **PromptTree > Selected: Add > Add from File System (Quick)** from the menu
3. Import starts with the previous settings

## Notes

- No settings dialog is displayed
- An error occurs if no previous settings are saved
- Convenient for repeatedly importing from the same folder
