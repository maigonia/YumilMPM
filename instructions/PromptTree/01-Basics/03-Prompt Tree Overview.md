# Prompt Tree Overview

## What Is This?

The prompt tree is a panel that hierarchically displays and manages [Prompt](01-Concepts/03-What%20Is%20a%20Prompt.md) materials within a [Category](../../AdvancedPanel/01-Basics/01-Category%20Template%20Tab.md).

It organizes materials (text fragments) used for prompt generation in a tree structure and selects which materials to include in generation using [Check State](01-Concepts/01-Understanding%20Check%20State.md).

## Main Roles

- **Material management**: Organize prompt materials in a [Hierarchy](01-Concepts/02-Understanding%20Hierarchy.md)
- **Generation target selection**: Select materials to include in generation with checkboxes
- **Content review**: Display the selected prompt's content in the [PromptEditor](../../PromptEditor/01-Basics/03-Prompt%20Editor.md)
- **[Search and Filtering](02-Operations/03-Search%20and%20Filtering.md)**: Narrow down materials by name, tag, or content

## Panel Structure

The prompt tree panel consists of the following parts:

1. **Header**: Panel name and add button
2. **Tree display area**: Hierarchical display of prompts
3. **Control bar**: Check operations and expand/collapse buttons
4. **Search panel**: Search box and filter options

See "[Panel Structure](04-Panel/03-PromptTree%20Panel%20Structure.md)" for details.

## Basic Operations

- **Click**: Select a prompt (content is displayed in the editor)
- **Checkbox**: Toggle whether to include in generation
- **Arrow icon**: Expand/collapse prompts with descendants
- **Drag & Drop**: Move a prompt to a different parent
- **Right-click**: Display context menu
