# Organizing and Moving

## Overview

This explains how to organize prompts for better visibility and management.

## Moving and Reordering

### Drag & Drop

Drag a prompt to move it under a different parent.

**Note**: Reordering between siblings (changing order within the same parent) has limitations with drag & drop.

### Up/Down Movement (Reordering)

Move the selected prompt one position at a time.

| Operation | Method |
| --- | --- |
| **Move Up** | Right-click → **Selected > Move Up** |
| **Move Down** | Right-click → **Selected > Move Down** |

### Copy & Paste

| Shortcut | Operation |
| --- | --- |
| **Ctrl+C** | Copy |
| **Ctrl+X** | Cut |
| **Ctrl+V** | Paste |

When pasting, if a prompt with the same name exists, it is automatically renamed.

### Batch Move of Checked Items

Checked prompts can be moved together.

1. Check the prompts to move
2. Select the destination prompt
3. Right-click → **Checked > Move under Selected**
4. Paste (Ctrl+V)

## Renaming

Change a prompt's name.

**Steps**:

1. Select the prompt
2. Press the **F2** key (or right-click → **Selected > Rename**)
3. Enter the new name
4. **Enter** to confirm (**Esc** to cancel)

### AI Name Suggestion

You can also have AI suggest names.

- **Selected > Suggest Name with AI...**: Suggest a name for the selected prompt
- **Checked > Bulk Suggest Names with AI...**: Bulk suggest names for checked prompts

## Deletion

### Individual Deletion

1. Select the prompt
2. Press the **Delete** key (or right-click → **Selected > Delete Prompts**)
3. Confirm in the confirmation dialog

### Batch Deletion

1. Check the prompts to delete
2. Right-click → **Checked > Delete Prompts**
3. Review the preview and confirm

### Delete with Files

For file prompts, associated files can also be deleted.

- Right-click → **Selected > Delete Chunk and Files...**

## Changing Icons

You can change a prompt's type (icon).

1. Right-click the prompt
2. **Selected > Change Icon** → **Set as Normal Prompt** or **Set as Group**

## Organization Tips

- **Meaningful group names**: Give names that indicate what's inside
- **Hierarchy depth**: Too deep makes operations difficult (2-3 levels is recommended)
- **Naming conventions**: Decide on consistent naming rules
