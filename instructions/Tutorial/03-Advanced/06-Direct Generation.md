# Direct Generation

Instantly execute the prompt editor content as a template for the selected category.

## Understanding with Examples

When testing a new Category Identifier syntax:

1. Type `@@@_Character.Filter(tag=anime)_@@@` in `@@@_Background_@@@` in the editor
2. Select a category from the header dropdown
3. Click the "Direct Generation" button
4. Instantly see the generation result

Skip the normal generation flow (add queue pattern → execute) and immediately check test results.

## Access

Prompt Editor header > Category selection > Direct Generation button

## Use Cases

- Testing Category Identifier syntax
- Verifying action chains (Target, Filter, Pick, etc.)
- Trial and error with templates
- Experimenting with new generation patterns

## Key Points

- Editor content becomes the template directly
- Generated using the selected category's settings
- References to other categories (@@@_xxx_@@@) are resolved normally
