# Prompt Generation

There are three methods for prompt generation:

## Once

Generate only once.
Use: Testing, single generation

## Timer

Auto-generate at specified intervals.
- Enter seconds in the textbox next to the button
- Prompts are automatically generated at the specified interval

Use: Periodic batch generation, auto-add time-consuming generation results as children of specified node for later batch use (`PromptTree > Selected: Add > Add From Generated Output`)

## OnDemand

Generate in response to requests from external apps.
Use: ComfyUI/Stable Diffusion WebUI integration

## Accessing Generated Files

Access completed prompt files from:
`Prompt Generation > Open Prompt Folder`
