# Prompt Browser

`PromptBrowser > Show/Hide` or double-click a selected prompt

View and edit additional information associated with prompts.

## Manageable Information

- **Images**: Reference images or sample images related to prompts
- **Memos**: Add multiple memos (usage examples, notes, etc.)
- **Tags**: Tags for search and filtering

## Image Panel Controls

- Mouse scroll: Zoom in/out
- Middle mouse button (drag): Pan
- Double-click: Fullscreen view
- Right-click: Show context menu
- In fullscreen: Move mouse to bottom of screen to show controller

## Tag Features

- AI auto-tagging: `PromptTree > Selected > Suggest Tags with AI...`
- Manage tags across all categories: `GlobalSettings > Tag Manager`

## Metadata Extraction

Can read metadata (prompts, settings, etc.) from images generated by Stable Diffusion or ComfyUI.

*This feature does not support all formats
