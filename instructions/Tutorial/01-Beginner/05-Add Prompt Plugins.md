# Add Prompt Plugins

There are various ways to add prompts, and you can select which methods to use.

## What are Plugins

From the pre-defined add methods, enable the ones you want to use and they will appear in the `PromptTree > Selected: Add` menu.

## Available Add Methods

- Add Child
- Add Child Group
- Add with Custom Name
- Add Sibling
- Add Clone
- Add Root Group
- Add from Clipboard (Lines)
- Add from Clipboard (Whole)
- Add from Clipboard (Tags)
- Add from Text File (Lines)
- Add from Text File (Whole)
- Add from JSON File
- Add from Civitai URL
- Add from File System
- Add from File System (Quick)
- Add from WD14
- Add from LM Studio

## Settings

`PromptTree > Selected: Add > Add Prompt Plugin Settings`
Check the add methods you want to use, and they will appear in the menu.
