# Sample Project

This app includes a built-in sample project. It helps you understand how to build trees, use Category Identifiers, and get a feel for actual usage.

## How to Access

1. Close the current project and return to the portal screen
2. Click the "Sample Project" button on the left
3. Select a destination folder and open it

## What You Can Learn

- Examples of category and prompt tree construction
- Dynamic prompt references using Category Identifiers
- How to configure output categories
- How generation results are assembled

Opening the sample project first to get an overview will make the rest of the tutorials easier to follow.
