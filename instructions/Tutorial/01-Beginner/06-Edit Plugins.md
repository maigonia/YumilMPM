# Edit Plugins

You can select plugins to assist with prompt editing.

## What are Plugins

From the pre-defined edit functions, enable the ones you want to use and they will appear in the `PromptEditor > Edit SelectedText with Plugins` menu.

## Available Edit Functions

- Wrap in Weight Syntax (text:1) — wrap text in weight format
- Trim Whitespace Each Line — remove trailing whitespace from each line
- Create Dynamic Prompts from Lines — convert newlines to dynamic prompts format
- Normalize Prompt Format — normalize prompt formatting
- Remove Duplicate Lines — remove duplicate lines
- Remove Duplicate Tags — remove duplicate tags
- Sort Lines Alphabetically — sort lines A-Z
- 
- Remove Tags at Cursor — remove tags at cursor position
- Escape as JS String — escape for JavaScript string literals
- Use Name as Content — use prompt name as content
- Replace Text — search and replace text
- LMStudio Edit — edit with local LLM
- WD14 Tagger — extract tags from images
- Google Translate — translate text
- Create Yumil Parser Block — convert to ComfyUI parser block format
- Filter Tags Using Presets — filter tags with saved presets
- Programmable Edit — edit with code

## Settings

`GlobalSettings > Edit Plugin Settings`
Check the edit functions you want to use, and they will appear in the menu.
