# Show AdvancedPanel
**Show AdvancedPanel**

## When to Use
- When you want to access advanced features like category templates, favorite queries, and checked lists
- When you want to search across the entire project
- When you want to save and restore check states as presets

## What It Does
Toggles the visibility of the Advanced Panel. Check to show the panel, uncheck to hide it.

The Advanced Panel has two scopes:
- **Category**: Operations for the selected category (category template, favorite queries, favorite lists, checked list)
- **Global**: Operations for the entire project (global search, global favorite queries, check presets)

## How to Use
1. Check "Show AdvancedPanel" in the menu
2. The panel appears
3. Select again to hide the panel

## Notes
- The panel is displayed below the PromptTree
- Switch scopes using the "Category" / "Global" buttons
- Category scope requires a category to be selected
