# Get Focus for Global Search
**Get Focus**

## When to Use
- When you want to search for prompts across the entire project
- When you want to find prompts with specific keywords or tags from all categories

## What It Does
Opens the Advanced Panel in Global scope, switches to the Search All tab, and moves focus to the search text box.

In Global Search, you can search for prompts across all categories. Search results are displayed grouped by category.

## How to Use
1. Select "Get Focus" from the menu
2. The Advanced Panel appears with the Search All tab active
3. Focus moves to the search text box
4. Enter search criteria and execute the search

## Notes
- You can choose the search target (name, content, tag, path)
- Regex search, exclusion search, and date filters are available
- Clicking a search result navigates to the corresponding category and prompt
- You can change the check state directly from search results using checkboxes
