# Save Current State
**Save Current State**

## When to Use
- When you want to save the current check state (all categories) as a preset
- When you want to record a specific work-related check state

## What It Does
Opens a dialog to save the current project-wide check state (all checked prompts across all categories) as a preset.

Saved presets can be restored with a single click, making it convenient for switching between work modes.

## How to Use
1. Set the check state you want to save in the PromptTree
2. Select "Save Current State" from the menu
3. The preset editing dialog opens
4. Enter a preset name and description
5. Click "Save"

## Notes
- Category selection is not required (Global scope)
- The check state of all categories is saved at once
- Each category's "generation enabled" state is also saved
- The same operation is available from the right-click menu
