# Get Focus for Check Presets
**Get Focus**

## When to Use
- When you want to operate the Check Presets panel with the keyboard
- When you want to quickly apply a saved check preset

## What It Does
Opens the Advanced Panel in Global scope, switches to the Check Presets tab, and moves focus to the "Save Current State" button.

In the Check Presets tab, you can click saved check presets to restore the check state across the entire project at once.

## How to Use
1. Select "Get Focus" from the menu
2. The Advanced Panel appears with the Check Presets tab active
3. Focus moves to the "Save Current State" button
4. Use arrow keys to navigate the preset list, Enter to apply a preset

## Notes
- Category selection is not required (Global scope)
- Clicking a preset shows a confirmation dialog (if there are existing checks)
- Right-click for preset editing, deletion, and reordering
- Applying a preset overwrites the check state of all categories
