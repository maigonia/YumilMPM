# Edit
**Edit**

## When to Use
- When you want to change the name or description of a saved check preset

## What It Does
Opens an editing dialog for the currently selected check preset in Global scope.

You can change the preset name and description.

## How to Use
1. Select a preset in the Check Presets tab (arrow keys or click)
2. Select "Edit..." from the menu
3. The editing dialog opens
4. Modify the content and click "Save"

## Notes
- A preset must be selected
- The same operation is available from the right-click menu
