# Delete
**Delete**

## When to Use
- When you want to remove a check preset that is no longer needed

## What It Does
Deletes the currently selected check preset in Global scope.

A confirmation dialog is shown before deletion.

## How to Use
1. Select a preset in the Check Presets tab (arrow keys or click)
2. Select "Delete" from the menu
3. Click "OK" in the confirmation dialog

## Notes
- A preset must be selected
- Deletion is not finalized until you save the project
- The same operation is available from the right-click menu
