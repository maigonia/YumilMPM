# Get Focus for Category Template
**Get Focus**

## When to Use
- When you want to operate the Category Template panel with the keyboard
- When you want to quickly change template settings

## What It Does
Opens the Advanced Panel in Category scope, switches to the Category Template tab, and moves focus to the Parts radio button.

In Category Template, you can manage per-category generation template settings (Target, Filter, Pick, Sort, Join, Edit).

## How to Use
1. Select "Get Focus" from the menu
2. The Advanced Panel appears with the Category Template tab active
3. Focus moves to the Parts radio button
4. Press Tab to navigate between combo boxes and change settings

## Notes
- A category must be selected
- You can switch between Parts mode and All mode
- The Clear button resets all settings to defaults
- Final Edit allows you to configure post-generation processing
