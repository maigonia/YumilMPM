# Pick ComboBox List Setting
**Pick**

## When to Use
- When you want to edit the choices in the Pick combo box
- When you want to add frequently used Pick settings to the list

## What It Does
Opens a dialog to edit the list items displayed in the Category Template's Pick combo box.

Pick is a setting that specifies how to select from multiple candidates during generation.

## How to Use
1. Select "Pick" from the menu
2. The list editing dialog opens
3. Add, edit, delete, or reorder items
4. Click "OK" to save

## Notes
- The first item is used as the default value
- Each category can have its own list
