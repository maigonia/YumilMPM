# Target ComboBox List Setting
**Target**

## When to Use
- When you want to edit the choices in the Target combo box
- When you want to add frequently used Target settings to the list

## What It Does
Opens a dialog to edit the list items displayed in the Category Template's Target combo box.

Target is a setting that specifies which prompts to target during generation.

## How to Use
1. Select "Target" from the menu
2. The list editing dialog opens
3. Add, edit, delete, or reorder items
4. Click "OK" to save

## Notes
- The first item is used as the default value
- Each category can have its own list
