# All ComboBox List Setting

**All**

## When to Use

- When you want to edit the choices in the All combo box
- When you want to add frequently used template directives to the list

## What It Does

Opens a dialog to edit the list items displayed in the Category Template's All combo box.

All is a setting that specifies the template directive to use when the template mode is "All".

## How to Use

1. Select "All" from the menu
2. The list editing dialog opens
3. Add, edit, delete, or reorder items
4. Click "OK" to save

## Notes

- The first item is used as the default value
- Each category can have its own list
