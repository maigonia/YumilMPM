# Final Edit ComboBox List Setting

**Final Edit**

## When to Use

- When you want to edit the choices in the Final Edit combo box
- When you want to add frequently used Final Edit plugins to the list

## What It Does

Opens a dialog to edit the list items displayed in the Category Template's Final Edit combo box.

Final Edit is the setting that specifies which editing plugin to apply as the final step after combining all category generation results.

## How to Use

1. Select "Final Edit" from the menu
2. The list editing dialog opens
3. Add, edit, delete, or reorder items
4. Click "OK" to save

## Notes

- The first item is used as the default value
- Each category can have its own list
- The Final Edit checkbox must be enabled to use Final Edit
