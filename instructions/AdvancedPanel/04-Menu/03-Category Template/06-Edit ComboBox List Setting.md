# Edit ComboBox List Setting

**Edit**

## When to Use

- When you want to edit the choices in the Edit combo box
- When you want to add frequently used Edit plugins to the list

## What It Does

Opens a dialog to edit the list items displayed in the Category Template's Edit combo box.

Edit is the setting that specifies which editing plugin to apply to prompt text during generation.

## How to Use

1. Select "Edit" from the menu
2. The list editing dialog opens
3. Add, edit, delete, or reorder items
4. Click "OK" to save

## Notes

- The first item is used as the default value
- Each category can have its own list
