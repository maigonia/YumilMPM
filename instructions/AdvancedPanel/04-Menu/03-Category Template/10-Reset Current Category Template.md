# Reset Current Category Template
**Reset Current Category Template**

## When to Use
- When you want to reset the currently selected category's template settings to defaults
- When you want to start over after experimenting with template settings

## What It Does
Resets the currently selected category's template settings (Target, Filter, Pick, Sort, Join, Edit, Final Edit) to their default values.

## How to Use
1. Select the category you want to reset
2. Select "Reset Current Category Template" from the menu
3. A confirmation dialog appears
4. Click "OK" to execute the reset

## Notes
- A category must be selected
- This operation cannot be undone
- ComboBox list items themselves are preserved
