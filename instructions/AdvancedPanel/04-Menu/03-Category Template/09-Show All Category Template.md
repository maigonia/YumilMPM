# Show All Category Template
**Show All Category Template**

## When to Use
- When you want to see all category template settings across the project at a glance
- When you want to check which categories have which templates configured

## What It Does
Opens a dialog that displays all category template settings across the project in a list.

You can check each category's Target, Filter, Pick, Sort, Join, Edit, and Final Edit settings.

## How to Use
1. Select "Show All Category Template" from the menu
2. A dialog opens showing template settings for all categories in a list
3. Close the dialog when done reviewing

## Notes
- Useful for identifying which categories have modified settings
- This screen is view-only and does not allow editing
