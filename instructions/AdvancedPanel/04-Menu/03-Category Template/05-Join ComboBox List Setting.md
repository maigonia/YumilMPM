# Join ComboBox List Setting
**Join**

## When to Use
- When you want to edit the choices in the Join combo box
- When you want to add frequently used Join settings to the list

## What It Does
Opens a dialog to edit the list items displayed in the Category Template's Join combo box.

Join is a setting that specifies how multiple results are combined during generation.

## How to Use
1. Select "Join" from the menu
2. The list editing dialog opens
3. Add, edit, delete, or reorder items
4. Click "OK" to save

## Notes
- The first item is used as the default value
- Each category can have its own list
