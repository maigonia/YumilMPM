# Reset All Category Template
**Reset All Category Template**

## When to Use
- When you want to reset all category template settings across the project at once
- When you want to reset template settings project-wide

## What It Does
Resets all category template settings (Target, Filter, Pick, Sort, Join, Edit, Final Edit) across the project to their default values.

## How to Use
1. Select "Reset All Category Template" from the menu
2. A confirmation dialog appears
3. Click "OK" to reset all category templates

## Notes
- This operation cannot be undone
- Affects all categories, so proceed with caution
- ComboBox list items themselves are preserved
