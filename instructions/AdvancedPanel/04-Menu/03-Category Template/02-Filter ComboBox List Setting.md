# Filter ComboBox List Setting
**Filter**

## When to Use
- When you want to edit the choices in the Filter combo box
- When you want to add frequently used Filter settings to the list

## What It Does
Opens a dialog to edit the list items displayed in the Category Template's Filter combo box.

Filter is a setting that specifies filtering conditions to apply during generation.

## How to Use
1. Select "Filter" from the menu
2. The list editing dialog opens
3. Add, edit, delete, or reorder items
4. Click "OK" to save

## Notes
- The first item is used as the default value
- Each category can have its own list
