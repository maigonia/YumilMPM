# Get Focus for Favorite Queries
**Get Focus**

## When to Use
- When you want to operate the Favorite Queries panel with the keyboard
- When you want to quickly execute a saved favorite query

## What It Does
Opens the Advanced Panel in Category scope, switches to the Favorite Queries tab, and moves focus to the "Add New" button.

In the Favorite Queries tab, you can click saved search criteria (favorite queries) to quickly execute searches.

## How to Use
1. Select "Get Focus" from the menu
2. The Advanced Panel appears with the Favorite Queries tab active
3. Focus moves to the "Add New" button
4. Use arrow keys to navigate the query list, Enter to execute a search

## Notes
- A category must be selected
- Clicking a query filters the PromptTree
- Right-click for query editing, deletion, and reordering
