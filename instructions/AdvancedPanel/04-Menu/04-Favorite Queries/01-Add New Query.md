# Add New Query
**Add New Query**

## When to Use
- When you want to save frequently used search criteria as a favorite
- When you want to register a recurring search pattern

## What It Does
Opens a dialog to add a new favorite query in Category scope.

A favorite query stores search criteria (query string, search target, regex option, exclusion search, etc.). You can execute the search with a single click.

## How to Use
1. Select "Add New Query" from the menu
2. The favorite query editing dialog opens
3. Enter the query name and search criteria
4. Click "Save"

## Notes
- A category must be selected
- Each category can have its own favorite query list
- The same operation is available from the right-click menu
