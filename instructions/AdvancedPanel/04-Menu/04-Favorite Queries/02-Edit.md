# Edit
**Edit**

## When to Use
- When you want to change the search criteria of a saved favorite query
- When you want to rename a query

## What It Does
Opens an editing dialog for the currently selected favorite query in Category scope.

You can change the query name, search criteria (query string, search target, regex option, etc.).

## How to Use
1. Select a query in the Favorite Queries tab (arrow keys or click)
2. Select "Edit..." from the menu
3. The editing dialog opens
4. Modify the content and click "Save"

## Notes
- A query must be selected
- Built-in queries cannot be edited
- The same operation is available from the right-click menu
