# Delete
**Delete**

## When to Use
- When you want to remove a favorite query that is no longer needed

## What It Does
Deletes the currently selected favorite query in Category scope.

A confirmation dialog is shown before deletion.

## How to Use
1. Select a query in the Favorite Queries tab (arrow keys or click)
2. Select "Delete" from the menu
3. Click "OK" in the confirmation dialog

## Notes
- A query must be selected
- Built-in queries cannot be deleted
- Deletion is not finalized until you save the project
- The same operation is available from the right-click menu
