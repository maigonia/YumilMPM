# Delete
**Delete**

## When to Use
- When you want to remove a favorite list that is no longer needed

## What It Does
Deletes the currently selected favorite list in Category scope.

A confirmation dialog is shown before deletion.

## How to Use
1. Select a list in the Favorite Lists tab (arrow keys or click)
2. Select "Delete" from the menu
3. Click "OK" in the confirmation dialog

## Notes
- A list must be selected
- Deletion is not finalized until you save the project
- The same operation is available from the right-click menu
