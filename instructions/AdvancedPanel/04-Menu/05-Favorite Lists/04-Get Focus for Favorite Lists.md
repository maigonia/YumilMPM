# Get Focus for Favorite Lists
**Get Focus**

## When to Use
- When you want to operate the Favorite Lists panel with the keyboard
- When you want to quickly apply a saved favorite list

## What It Does
Opens the Advanced Panel in Category scope, switches to the Favorite Lists tab, and moves focus to the "Create New" button.

In the Favorite Lists tab, you can toggle the check state of prompts in the PromptTree by checking/unchecking saved prompt lists.

## How to Use
1. Select "Get Focus" from the menu
2. The Advanced Panel appears with the Favorite Lists tab active
3. Focus moves to the "Create New" button
4. Use arrow keys to navigate lists, Enter to toggle check state

## Notes
- A category must be selected
- Clicking a list toggles the check state of prompts in the PromptTree
- Right-click for list editing, deletion, and reordering
