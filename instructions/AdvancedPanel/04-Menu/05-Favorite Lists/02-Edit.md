# Edit
**Edit**

## When to Use
- When you want to change the name of a saved favorite list
- When you want to update the contents of a list

## What It Does
Opens an editing dialog for the currently selected favorite list in Category scope.

You can change the list name and the prompt ID list it contains.

## How to Use
1. Select a list in the Favorite Lists tab (arrow keys or click)
2. Select "Edit..." from the menu
3. The editing dialog opens
4. Modify the content and click "Save"

## Notes
- A list must be selected
- The same operation is available from the right-click menu
