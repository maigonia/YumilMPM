# Create New List

**Create New List**

## When to Use

- When you want to save the currently checked prompts as a list
- When you want to register a frequently used combination of prompts

## What It Does

Opens a dialog to create a new favorite list in Category scope.

A favorite list stores a list of prompt IDs. By clicking the checkbox, you can batch toggle the check state of prompts in the PromptTree.

## How to Use

1. Check the prompts you want to save in the PromptTree
2. Select "Create New List" from the menu
3. The list creation dialog opens
4. Enter a list name and click "Create"

## Notes

- A category must be selected
- Each category can have its own favorite lists
- The currently checked prompts are included in the list
