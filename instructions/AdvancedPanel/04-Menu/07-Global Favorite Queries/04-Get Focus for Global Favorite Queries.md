# Get Focus for Global Favorite Queries
**Get Focus**

## When to Use
- When you want to operate the Global Favorite Queries panel with the keyboard
- When you want to quickly execute a saved global favorite query

## What It Does
Opens the Advanced Panel in Global scope, switches to the Favorite Queries tab, and moves focus to the "Add New" button.

In the Global Favorite Queries tab, you can click saved global search criteria to quickly execute cross-category searches.

## How to Use
1. Select "Get Focus" from the menu
2. The Advanced Panel appears with the Favorite Queries tab active
3. Focus moves to the "Add New" button
4. Use arrow keys to navigate the query list, Enter to execute a search

## Notes
- Category selection is not required (Global scope)
- Clicking a query switches to the Global Search tab and runs the search
- Right-click for query editing, deletion, duplication, and reordering
