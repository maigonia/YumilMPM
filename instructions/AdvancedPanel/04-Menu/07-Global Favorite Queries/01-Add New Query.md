# Add New Query
**Add New Query**

## When to Use
- When you want to save search criteria for use across the entire project as a favorite
- When you want to register a recurring global search pattern

## What It Does
Opens a dialog to add a new favorite query in Global scope.

A global favorite query stores search criteria at the project level. You can execute a Global Search (cross-category search) with a single click.

## How to Use
1. Select "Add New Query" from the menu
2. The favorite query editing dialog opens
3. Enter the query name and search criteria
4. Click "Save"

## Notes
- Category selection is not required (Global scope)
- The same operation is available from the right-click menu
- Executing a query switches to the Global Search tab and runs the search
