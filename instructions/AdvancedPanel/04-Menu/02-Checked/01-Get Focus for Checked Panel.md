# Get Focus for Checked Panel
**Get Focus**

## When to Use
- When you want to operate the Checked panel with the keyboard
- When you want to quickly review the list of checked prompts

## What It Does
Opens the Advanced Panel in Category scope, switches to the Checked tab, and moves focus to the "Clear All" button.

The Checked tab shows a list of prompts that are currently checked in this category.

## How to Use
1. Select "Get Focus" from the menu
2. The Advanced Panel appears with the Checked tab active
3. Focus moves to the "Clear All" button
4. Use arrow keys to navigate the list, Enter to jump to the corresponding prompt in the tree

## Notes
- A category must be selected
- The Checked tab also supports bulk editing (Bulk Edit) of checked prompts
- Clicking a prompt shows the corresponding item in the PromptTree
