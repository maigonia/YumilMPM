# Switching Work Modes with Check Presets

By saving multiple "work modes" as [Check Preset](../02-Glossary/02-Check Preset.md)s, you can switch between them with a single click.

## Example: Switching Between Anime Style / Photo Style

When the same project contains prompts for different styles, re-checking each time is tedious.

### Preparation

1. Check all prompts you want for anime-style generation across all categories
2. In the **Global > Check Presets** tab, click **Save Current State**
3. Save with the preset name "Anime Style"
4. Next, re-check prompts for photo style
5. Save similarly as "Photo Style"

### How to Use

Just click "Anime Style" in the Check Presets tab, and the check states across all categories will switch.

## Combining with Category-Level Lists

Using [Check Preset](../02-Glossary/02-Check Preset.md) for project-wide switching and [Favorite List](../02-Glossary/03-Favorite List.md) for fine-grained combinations within categories is efficient.

**Example**: After applying the "Anime Style" preset, if you want to change only the LoRA category to a different combination, switch the LoRA category's list in Category scope Favorite Lists.
