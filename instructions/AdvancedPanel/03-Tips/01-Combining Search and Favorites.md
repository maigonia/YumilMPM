# Combining Search and Favorites

Here are some ways to efficiently search for prompts by combining [Favorite Query](../02-Glossary/04-Favorite%20Query.md) and Global Search.

## Save Frequently Used Searches as Favorites

There's no need to enter the same search criteria repeatedly.

1. Set search criteria in Global Search or the Favorite Queries tab
2. Select "Add New" from the right-click menu to save as a favorite
3. Next time, just click to execute the search instantly

## Using Exclusion Search

To find prompts that do **not** have a specific tag, use the Exclude option.

**Example**: Finding prompts that haven't been tagged yet
- Search target: Tag
- Search text: Any tag name
- Exclude: ON

## Check Recent Work with Date Filters

Using Global Search's date filter, you can quickly find recently added or modified prompts. You can search with just a date filter even if the search text is empty.

**Example**: Check prompts added this week
- Date Filter: This Week
- Search text: (leave empty)

## Choosing Between Global Search and Category Favorite Queries

| Scenario | Recommended |
|----------|-------------|
| Don't know which category it's in | Global Search |
| Want to filter within a specific category | Category Favorite Queries |
| Want to check prompts directly from results | Global Search (has checkboxes) |
| Want to filter the PromptTree display | Category Favorite Queries (tree filter) |
