# Global Search (Search All) Tab

This is a tab for searching prompts across the entire project, displayed in Global scope.

## When Is It Useful

- When you want to find prompts with specific keywords or tags from all categories at once
- When you don't know which category the target prompt is in
- When you want to filter recently added or modified prompts by date

## Search Options

| Item | Description |
|------|-------------|
| Search text | Keywords to search for |
| Search target (Target Type) | Name / Content / Tag / Path |
| Regex | Use regular expressions |
| Exclude | Show items that do "not" match the criteria |
| Date Filter | Filter by date |

### Date Filter Types

| Filter | Description |
|--------|-------------|
| Today | Updated today |
| Yesterday | Updated yesterday |
| Last 7 Days | Past 7 days |
| Last 30 Days | Past 30 days |
| This Week | This week |
| Last Week | Last week |
| This Month | This month |
| Last Month | Last month |

You can search with just a Date Filter even if the search text is empty.

## Search Results Display

Search results are displayed grouped by category. Each category group can be collapsed/expanded.

Each result shows the following:
- Checkbox (click to toggle check state)
- Icon (folder / file)
- Prompt name

## Operations from Search Results

| Operation | Result |
|-----------|--------|
| Click prompt name | Navigate to that category and show the prompt in PromptTree |
| Click checkbox | Toggle check state (without changing category) |
| Category name ▶ / ▼ | Expand/collapse the group |
