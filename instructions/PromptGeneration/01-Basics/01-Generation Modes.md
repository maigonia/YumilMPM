# Generation Modes

There are 3 modes for prompt generation. Choose the one that fits your needs.

## Once Generation (Manual Generation)

Generates a prompt once each time you press the button. The most basic usage.

See [Once Generation](../04-Menu/01-Once%20Generation.md) for details.

## Timer Generation

Automatically generates prompts at a set interval (in seconds). Generation continues until stopped.

Useful when you want to create many prompts at once or need different prompts in succession.

See [Timer Generation Start](../04-Menu/03-Timer%20Generation/01-Timer%20Generation%20Start.md) for details.

## Generation On Demand

Works with external apps like ComfyUI or Stable Diffusion WebUI to automatically generate prompts triggered by the external app's operations.

Each time you press the "Generate" button in the image generation AI, this app automatically supplies a new prompt, eliminating the need to manually paste prompts.

See [Start On Demand](../04-Menu/02-Generation%20On%20Demand/01-Start%20On%20Demand.md) for details. For external app configuration, see [ComfyUI Extension](05-External%20App%20Integration/01-ComfyUI%20Extension.md) and [Stable Diffusion Extension](05-External%20App%20Integration/02-Stable%20Diffusion%20Extension.md).

## Mode Comparison

| Mode | Operation | Use Case |
|------|-----------|----------|
| Once Generation | Manual, one at a time | Checking prompts, single generation |
| Timer Generation | Automatic repetition | Mass generation, continuous generation |
| Generation On Demand | Linked with external apps | Integration with image generation AI |
