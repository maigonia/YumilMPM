# ResultWindow Always on Top

**ResultWindow Always on Top**

## When to Use

- When you want the result window to stay visible above other windows
- When you want to keep generation results visible while working with image generation software

## What It Does

Configures whether the result window is always displayed on top of other windows. When checked, the result window stays in the foreground at all times.

## How to Use

1. Select **PromptGeneration > ResultWindow Always on Top** from the menu
2. Toggle with the checkmark on/off

## Notes

- Disabled by default
- If the result window is not open, the setting will be applied the next time the window is opened
- If the result window is already open, the change takes effect immediately
