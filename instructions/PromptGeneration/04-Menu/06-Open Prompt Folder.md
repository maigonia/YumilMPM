# Open Prompt Folder

**Open Prompt Folder**

## When to Use

- When you want to check the text files of generated prompts
- When you want to edit saved prompts with an external app
- When you want to verify the prompt save destination folder

## What It Does

Opens the folder where generation results are saved in the OS file explorer.

## How to Use

1. Select **PromptGeneration > Open Prompt Folder** from the menu
2. The prompt folder opens in the explorer

## Notes

- When "Auto-save prompts to text files" is enabled, generation results are saved in this folder
- The folder is automatically created if it doesn't exist
