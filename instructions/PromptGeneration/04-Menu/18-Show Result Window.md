# Show Result Window

**Show Result Window**

## When to Use

- When you want to manually open the generation result window
- When you want to reopen a closed result window
- When you want to check previous generation results

## What It Does

Opens the window that displays prompt generation results. If results have been generated, you can view their contents.

## How to Use

1. Select **PromptGeneration > Show Result Window** from the menu
2. The result window appears

## Notes

- An empty window is shown if no generation results exist
- You can copy prompts from the result window
