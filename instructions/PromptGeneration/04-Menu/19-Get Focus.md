# Get Focus for Generation Panel

**Get Focus**

## When to Use

- When you want to operate the Generation panel with the keyboard
- When you want to move from another panel to the Generation panel

## What It Does

Moves keyboard focus to the "Once" button in the Generation panel. After focusing, you can use the Tab key to navigate through each button and input field in the panel.

## How to Use

Select **PromptGeneration > Get Focus** from the menu

## Tips

- Alt+click the menu item to assign a shortcut for quicker access
- After focusing, Tab moves through: Once → Demand → Timer → Timer interval input → Queue
