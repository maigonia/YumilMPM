# Auto-save prompts to text files

**Auto-save prompts to text files**

## When to Use

- When you want generated prompts to be automatically saved as files
- When you want to keep a history of generated files
- When you want to use prompts with external tools

## What It Does

Configures whether generation results are automatically saved as text files after prompt generation. When checked, a text file is created in the prompt folder with each generation.

## How to Use

1. Select **PromptGeneration > Auto-save prompts to text files** from the menu
2. Toggle with the checkmark on/off

## Notes

- Check the save location with "Open Prompt Folder"
- File names include the generation date and time
