# Auto-Send Result to Clipboard

**Auto-Send Result to Clipboard**

## When to Use

- When you want to paste generated prompts into other apps right away
- When you want to pass prompts to external tools like image generation AI
- When you want to skip copy operations for efficiency

## What It Does

Configures whether generation results are automatically copied to the clipboard after prompt generation. When checked, prompts are copied to the clipboard with each generation.

## How to Use

1. Select **PromptGeneration > Auto-Send Result to Clipboard** from the menu
2. Toggle with the checkmark on/off

## Notes

- When enabled, you can paste with Ctrl+V immediately after generation
- Existing clipboard contents will be overwritten
- During loop generation, the last generated prompt remains in the clipboard
