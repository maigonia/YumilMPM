# Enable CommentOut

**Enable CommentOut**

## When to Use

- When you want to include comments in the generation result
- When you want to keep debug information in the output

## What It Does

Configures whether comments (lines starting with `//`, etc.) are included in generation results. When checked, comment lines are also output.

## How to Use

1. Select **PromptGeneration > Enable CommentOut** from the menu
2. Toggle with the checkmark on/off

## Notes

- Enabled by default (comments are not output)
- When sending to image generation AI, it's recommended to leave this disabled
