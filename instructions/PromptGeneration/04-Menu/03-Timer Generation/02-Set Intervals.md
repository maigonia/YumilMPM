# Set Intervals

**Timer Generation > Set Intervals**

## When to Use

- When you want to change the timer generation interval
- When you want to auto-generate faster or slower

## What It Does

Sets the interval (in milliseconds) used for timer generation. This setting takes effect from the next timer generation start.

## How to Use

1. Select **PromptGeneration > Timer Generation > Set Intervals** from the menu
2. A dialog appears
3. Enter the interval in milliseconds (e.g., 5000 = 5 seconds)
4. Click OK to save

## Examples

- **5000**: Generate every 5 seconds
- **10000**: Generate every 10 seconds (default)
- **30000**: Generate every 30 seconds
- **60000**: Generate every 1 minute

## Notes

- The setting is saved to the project
- The value shown in the current menu is the current setting
- The interval can be changed during timer generation (takes effect from the next cycle)
