# Show ResultWindow After Generation

**Show ResultWindow After Generation**

## When to Use

- When you want to immediately check and copy generation results

## What It Does

Configures whether the result window is automatically displayed after prompt generation completes. When checked, the result window is automatically shown upon generation completion.

## How to Use

1. Select **PromptGeneration > Show ResultWindow After Generation** from the menu
2. Toggle with the checkmark on/off

## Notes

- Enabled by default
- When disabled, you need to manually open the result window from **Show Result Window**
- Can be used in combination with "Activate ResultWindow After Generation"
