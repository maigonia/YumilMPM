# Collapse All
**Collapse All**

## When to Use
- When you want a compact view of the category tree
- When you want to display only root categories

## What It Does
Collapses all categories in the category tree, showing only root categories.

## How to Use
Select **CategoryTree > Collapse All**

## Notes
- All categories with children are recursively collapsed
- The current selection state is maintained
