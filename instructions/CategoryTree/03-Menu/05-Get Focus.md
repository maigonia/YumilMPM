# Get Focus
**Get Focus**

## When to Use
- When you want to operate the category tree with the keyboard
- When you want to move to the category tree from another panel

## What It Does
Moves keyboard focus to the category tree. After focusing, you can navigate categories with arrow keys and use keyboard shortcuts.

## How to Use
Select **CategoryTree > Get Focus**

## Tips
- Alt+click a top menu item to assign a shortcut for quicker access
