# Add Root Category
**Add Root Category**

## When to Use
- When you want to create a new top-level category
- When you want to add a new classification to your project

## What It Does
Adds a new category at the top level (root level) of the category tree.

## How to Use
1. Select **CategoryTree > Add Root Category**
2. Enter the new category name in the dialog
3. Click **Add**

## Notes
- Category names must be unique within the project (case-insensitive)
- A "Root" prompt is automatically created in the new category
- Can be executed even without a category selected

## Tips
- You can also add from the **+ Category** button at the top of the tree
- The same operation is available from the context menu (right-click)
