# Expand All
**Expand All**

## When to Use
- When you want to expand all categories at once
- When you want to see the entire category tree structure

## What It Does
Expands all categories in the category tree, showing all hierarchy levels.

## How to Use
Select **CategoryTree > Expand All**

## Notes
- All categories with children are recursively expanded
- The display may become long if there are many categories
