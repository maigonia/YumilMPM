# Move Up
**Move Up**

## When to Use
- When you want to change the display order of categories
- When you want to move a category up

## What It Does
Moves the selected category one position up within the same hierarchy level.

## How to Use
Select **CategoryTree > Selected Category > Move Up**

## Notes
- Movement is within the same hierarchy level (between sibling categories)
- A category at the top cannot be moved further up
- You can also change the order by drag & drop
