# Add Child Category
**Add Child Category**

## When to Use
- When you want to create a subcategory under an existing category
- When you want to organize categories hierarchically
- When you want to further classify related prompts

## What It Does
Adds a new child category under the selected category. The added child category appears under the parent category.

## How to Use
1. Select the parent category in the category tree
2. Select **CategoryTree > Selected Category > Add Child Category**
3. Enter the new category name in the dialog
4. Click **Add**

## Notes
- Category names must be unique within the project (case-insensitive)
- A "Root" prompt is automatically created in the new category
- After adding a child category, the parent category is automatically expanded

## Tips
- Press **Enter** to confirm after entering the category name
- The same operation is available from the context menu (right-click)
