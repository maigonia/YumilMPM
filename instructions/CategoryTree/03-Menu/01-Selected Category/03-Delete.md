# Delete
**Delete**

## When to Use
- When you want to delete an unnecessary category
- When you want to reorganize the category structure

## What It Does
Deletes the selected category.

## How to Use
1. Select the category to delete in the category tree
2. Select **CategoryTree > Selected Category > Delete**
3. Click **OK** in the confirmation dialog

## Notes
- **Deleting a category also deletes all prompts within it**
- If there are child categories, the number of descendant categories is shown in the confirmation dialog
- After deletion, the selection state is cleared

## Tips
- You can also delete with the **Delete** key (when the category tree has focus)
- Check for important prompts before deleting
