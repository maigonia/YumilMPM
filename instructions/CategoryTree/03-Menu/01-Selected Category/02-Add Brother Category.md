# Add Brother Category
**Add Brother Category**

## When to Use
- When you want to add a category at the same level as the selected category
- When you want to create a classification parallel to an existing category

## What It Does
Adds a new category at the same level (under the same parent) as the selected category.

## How to Use
1. Select a category in the category tree
2. Select **CategoryTree > Selected Category > Add Brother Category**
3. Enter the new category name in the dialog
4. Click **Add**

## Notes
- Category names must be unique within the project (case-insensitive)
- A "Root" prompt is automatically created in the new category
- If a root category is selected, a new root category is added

## Tips
- The same operation is available from the context menu (right-click)
