# Rename
**Rename**

## When to Use
- When you want to change a category's name
- When you want to correct the name to something more descriptive

## What It Does
Changes the name of the selected category.

## How to Use
1. Select a category in the category tree
2. Select **CategoryTree > Selected Category > Rename**
3. Enter the new name in the dialog
4. Click **OK**

## Notes
- Category names must be unique within the project (case-insensitive)
- Prompts within the category are preserved after renaming

## Tips
- You can also rename with the **F2** key (when the category tree has focus)
- The same operation is available from the context menu (right-click)
