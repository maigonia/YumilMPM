# Basic Editing

Basic text editing operations available in the prompt editor.

## Operation List

| Operation | Menu | Shortcut |
|-----------|------|----------|
| Copy | PromptEditor > Copy | Ctrl+C |
| Cut | PromptEditor > Cut | Ctrl+X |
| Paste | PromptEditor > Paste | Ctrl+V |
| Select All | PromptEditor > Select All | Ctrl+A |

*On Mac, use Cmd instead of Ctrl.
