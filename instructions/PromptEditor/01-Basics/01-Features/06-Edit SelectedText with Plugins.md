# Edit SelectedText with Plugins

**Edit SelectedText with Plugins**

## What It Does

Processes text currently selected in the editor with plugins. Perform various editing operations like text formatting, conversion, and duplicate removal with a single click.

## How to Use

1. Select the text you want to process in the editor
2. Select **PromptEditor > Edit SelectedText with Plugins** from the menu
3. Click the plugin you want to use
4. The selected text is processed

## Plugin Display Settings

Plugins shown in the menu can be configured in **GlobalSettings > EditPlugin Settings**. Hiding unnecessary plugins keeps the menu clean.
