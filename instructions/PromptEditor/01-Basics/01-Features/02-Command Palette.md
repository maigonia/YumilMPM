# Command Palette

A command list accessible with the **F1** key for keyboard-driven operation. **F1** works from anywhere in the app, regardless of focus.

## When to Use

- When you want to operate without opening menus, using only the keyboard
- When you want to quickly find and execute a desired function

## What It Does

Search and execute various app functions by entering keywords. Complete operations with just the keyboard instead of menu navigation or button clicks.

## How to Use

1. Press the **F1** key (works from anywhere in the app)
2. The command palette appears
3. Enter keywords to narrow down the desired operation
4. Select the target command from the list to execute

## Category Prefixes

Entering a category name in the input field narrows down to commands in that category only.

| Prefix | Content |
|--------|---------|
| `editor:` | Editor operations |
| `gen:` | Generation-related |
| `file:` | File operations |
| `tree:` | Tree operations |
| `cat:` | Category operations |
| `browser:` | Browser operations |
| `settings:` | Settings |
| `util:` | Utilities |
