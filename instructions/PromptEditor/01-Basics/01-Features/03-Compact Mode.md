# Compact Mode

A feature that switches to a small window showing only the editor.

## When to Use

- When you want to work side by side with other applications
- When you want to focus only on prompt input
- When you want to quickly input prompts and generate

## What It Does

Hides the menu bar, trees, browser, etc. and allows working in a small window with only the editor portion. Category switching, generation, and copying are available from the footer.

## Basic Flow

1. Select **PromptEditor > Compact Mode** to enter compact mode
2. Edit in the small editor-only window
3. Click the **Gen** button in the footer to generate, **Copy** button to copy
4. Return to normal mode with the **maximize button** in the footer or from the menu

Window size can be freely changed and is automatically remembered for future sessions.

For detailed usage, see [Compact Mode](../../04-Menu/14-Compact Mode.md).
