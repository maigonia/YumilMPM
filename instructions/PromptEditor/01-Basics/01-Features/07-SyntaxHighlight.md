# SyntaxHighlight

A feature for visually highlighting text in the prompt editor.

## When to Use

- **When you want Category Identifiers to stand out**: Color-code `@@@_CategoryName_@@@`
- **When you want to highlight important keywords**: Display `IMPORTANT`, `TODO`, etc. in prominent colors
- **When you want to distinguish comment lines**: Display lines starting with `#` in different colors
- **When you want to identify specific tags**: Color `<tag>` format text

## What It Does

Highlight text matching user-defined patterns (strings or regular expressions) in your chosen colors. Create multiple rules and adjust their priority to visually clarify prompt structure.

## Basic Flow

1. **Enable**: Turn on the feature from the menu → [SyntaxHighlight Enable](../../04-Menu/11-SyntaxHighlight/01-Enable%20SyntaxHighlight.md)
2. **Configure**: Create and edit rules → [SyntaxHighlight Setting](../../04-Menu/11-SyntaxHighlight/02-SyntaxHighlight%20Setting.md)
3. **Verify**: Check the coloring in the editor
4. **Adjust**: Add or modify rules as needed

For detailed usage and configuration, see the links above.
