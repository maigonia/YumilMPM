# Keyboard Shortcuts

Keyboard shortcuts available in the prompt editor.

## Basic Operations

| Operation | Windows | Mac |
|-----------|---------|-----|
| Copy | Ctrl+C | Cmd+C |
| Cut | Ctrl+X | Cmd+X |
| Paste | Ctrl+V | Cmd+V |
| Select All | Ctrl+A | Cmd+A |
| Undo | Ctrl+Z | Cmd+Z |
| Redo | Ctrl+Y | Cmd+Shift+Z |

## Find & Replace

| Operation | Windows | Mac |
|-----------|---------|-----|
| Find | Ctrl+F | Cmd+F |
| Replace | Ctrl+H | Cmd+H |

## Other

| Operation | Windows | Mac |
|-----------|---------|-----|
| Command Palette (app-wide) | F1 | F1 |

## Customizing Shortcuts

By default, only basic operations are enabled. To enable additional shortcuts (multi-cursor, line operations, etc.), configure them in [Monaco Keybinding Settings](../04-Menu/13-Monaco Keybinding Settings.md).
