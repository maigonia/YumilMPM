# Prompt Editor

A text editor for editing prompt content.

## What It Does

- Create and edit prompts
- Text search and replace
- Basic operations like copy, cut, and paste
- Input assistance with autocomplete
- Text processing with plugins
- Operations from the command palette
- Focused editing in compact mode
- Quick output testing with direct generation
