# Find

**Find**

## When to Use

- When you want to find specific text within a prompt
- When you want to check where a specific word is used

## What It Does

Searches for text within the editor. A search bar appears and highlights locations matching the input string.

## How to Use

1. Select **PromptEditor > Find** from the menu
2. The search bar appears
3. Enter the text you want to search for
4. Matching locations are highlighted
5. Use the up/down arrows to navigate to the next/previous match

## Shortcut

- **Ctrl+F** (Windows)
- **Cmd+F** (Mac)

## Search Options

- **Case sensitive**: Aa icon
- **Whole word match**: Ab icon
- **Regular expression**: .* icon
