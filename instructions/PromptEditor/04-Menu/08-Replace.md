# Replace

**Replace**

## When to Use

- When you want to replace text with different text
- When you want to change the same text in multiple places at once

## What It Does

Searches for text in the editor and replaces it with different text. You can replace one at a time or all at once.

## How to Use

1. Select **PromptEditor > Replace** from the menu
2. The search/replace bar appears
3. Enter the text to search for
4. Enter the replacement text
5. Click **Replace** to replace one, or **Replace All** to replace all

## Shortcut

- **Ctrl+H** (Windows)
- **Cmd+H** (Mac)

## Replace Options

- **Case sensitive**: Aa icon
- **Whole word match**: Ab icon
- **Regular expression**: .* icon
