# Selected Text As Search Query

**Selected Text as Search Query**

## When to Use

- When you want to search for prompts using a tag or word selected in the editor

## What It Does

Uses the text currently selected in the editor to send it directly to the PromptTree search panel and execute filtering immediately.

## How to Use

### Basic Usage

1. **Select text**:
   - Drag to select the text you want to search for in the prompt editor

2. **Execute the function**:
   - Select **PromptEditor > Selected Text As Search Query** from the menu

3. **Check the results**:
   - The selected text is automatically entered in the ChunkTree search panel
   - The ChunkTree is filtered to prompts containing the selected text
   - Only matching prompts are displayed in the tree

### Execution Methods

1. **From the menu**:
   - **PromptEditor > Selected Text As Search Query**

2. **From the command palette** (F1):
   - `editor: Selected Text As Search Query`
