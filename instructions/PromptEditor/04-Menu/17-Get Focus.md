# Get Focus

**Get Focus**

## When to Use

- When you want to start typing in the editor immediately after other operations
- When you want to move focus to the editor with the keyboard

## What It Does

Moves keyboard focus to the prompt editor. This allows you to start typing immediately.

## How to Use

1. Select **PromptEditor > Get Focus** from the menu
2. Focus moves to the editor and the cursor blinks
3. You can start typing immediately

## Notes

- Setting a shortcut key allows you to quickly focus the editor without using the mouse
- The cursor position after focusing maintains the previous editing position
