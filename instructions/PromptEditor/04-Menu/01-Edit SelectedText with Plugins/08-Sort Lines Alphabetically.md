# Sort Lines Alphabetically

**Sort Lines Alphabetically**

## What It Does

Sorts each line of selected text in alphabetical (lexicographic) order.

## How to Use

1. Select the text to sort
2. Select **Edit SelectedText with Plugins > Sort Lines Alphabetically** from the menu
3. Lines are sorted alphabetically

## Example

- Input:
```
red hair
blue eyes
smile
```
- Output:
```
blue eyes
red hair
smile
```

## Notes

- Sorts case insensitively
