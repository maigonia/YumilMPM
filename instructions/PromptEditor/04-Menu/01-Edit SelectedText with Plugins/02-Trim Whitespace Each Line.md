# Trim Whitespace Each Line

**Trim Whitespace Each Line**

## What It Does

Removes leading and trailing whitespace (spaces, tabs) from each line of selected text.

## How to Use

1. Select the text to format
2. Select **Edit SelectedText with Plugins > Trim Whitespace Each Line** from the menu
3. Leading and trailing whitespace is removed from each line

## Example

- Input:
```
  red hair
    blue eyes
```
- Output:
```
red hair
blue eyes
```
