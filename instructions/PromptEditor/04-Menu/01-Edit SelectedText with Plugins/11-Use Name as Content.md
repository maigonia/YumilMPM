# Use Name as Content

**Use Name as Content**

## What It Does

Sets the prompt (chunk) name directly as its content. Useful when you want the content to be the same text as the prompt name.

## How to Use

1. Select **Edit SelectedText with Plugins > Use Name as Content** from the menu
2. The editor content is replaced with the prompt's name

## Example

- If the prompt name is `red hair`
- Content after execution: `red hair`

## Notes

- This feature can be combined with the chunk tree's bulk editing feature to apply to multiple prompts at once
