# Enable AutoComplete

**AutoComplete > Enable**

## When to Use

- When you want to toggle the [AutoComplete](../../01-Basics/01-Features/01-AutoComplete.md) feature on/off
- When you want to temporarily disable autocomplete

## What It Does

Toggles the [AutoComplete](../../01-Basics/01-Features/01-AutoComplete.md) feature on/off in the prompt editor.

## How to Use

1. Select **PromptEditor > AutoComplete > Enable** from the menu
2. Checked state → AutoComplete is enabled
3. Unchecked state → AutoComplete is disabled

## Notes

- The setting is maintained after app restart
- For detailed specifications and operation methods of autocomplete, see [AutoComplete](../../01-Basics/01-Features/01-AutoComplete.md)
