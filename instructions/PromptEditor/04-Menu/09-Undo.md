# Undo

**Undo**

## When to Use

- When you accidentally deleted text
- When you want to revert editing changes

## What It Does

Cancels the most recent editing operation and reverts to the previous state. Executing multiple times reverts further back.

## How to Use

1. Select **PromptEditor > Undo** from the menu
2. The most recent edit is cancelled

## Shortcut

- **Ctrl+Z** (Windows)
- **Cmd+Z** (Mac)

## Notes

- There is a limit to how many operations can be undone
- Switching prompts resets the undo history
