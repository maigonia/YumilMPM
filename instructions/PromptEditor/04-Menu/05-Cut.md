# Cut

**Cut**

## When to Use

- When you want to move selected text to another location
- When you want to delete text while saving it to the clipboard

## What It Does

Copies the text selected in the editor to the clipboard and removes it from the original location.

## How to Use

1. Select the text you want to cut
2. Select **PromptEditor > Cut** from the menu
3. The selected text is copied to the clipboard and removed from the editor

## Shortcut

- **Ctrl+X** (Windows)
- **Cmd+X** (Mac)
