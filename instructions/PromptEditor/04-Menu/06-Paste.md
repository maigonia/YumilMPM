# Paste

**Paste**

## When to Use

- When you want to insert copied or cut text
- When you want to paste text from another application

## What It Does

Inserts clipboard content at the cursor position in the editor. If text is selected, it replaces the selection.

## How to Use

1. Move the cursor to the insertion point (or select the part to replace)
2. Select **PromptEditor > Paste** from the menu
3. Clipboard content is inserted

## Shortcut

- **Ctrl+V** (Windows)
- **Cmd+V** (Mac)
