# Select All

**Select All**

## When to Use

- When you want to copy the entire prompt
- When you want to replace all text with different content

## What It Does

Selects all text in the editor.

## How to Use

1. Select **PromptEditor > Select All** from the menu
2. All text in the editor is selected

## Shortcut

- **Ctrl+A** (Windows)
- **Cmd+A** (Mac)
