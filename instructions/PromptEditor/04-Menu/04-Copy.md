# Copy

**Copy**

## When to Use

- When you want to copy selected text to the clipboard
- When you want to use the same text in multiple places

## What It Does

Copies the text selected in the editor to the clipboard. The original text remains unchanged.

## How to Use

1. Select the text you want to copy
2. Select **PromptEditor > Copy** from the menu
3. The selected text is copied to the clipboard

## Shortcut

- **Ctrl+C** (Windows)
- **Cmd+C** (Mac)
