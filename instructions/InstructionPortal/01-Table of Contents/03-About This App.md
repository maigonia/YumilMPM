# About This App

## What Is Yumil MPM

Yumil MPM (MainPromptMaker) is a prompt management tool.

It organizes large numbers of prompts (text fragments) in a tree structure, lets you select and combine them through check operations, and automatically generates the final prompt text.

## Key Features

| Feature | Description |
|---------|-------------|
| **Tree Management** | Organize prompts in a folder/file-style hierarchy |
| **Category Classification** | Create categories by purpose (e.g., Quality, Character, Background) |
| **Check Selection** | Select prompts to use via checkboxes |
| **Dynamic Text Substitution** | Cross-category references and dynamic prompt building via [Category Identifier](../../PromptEditor/02-How%20To%20Write%20Prompt%20Content/01-How%20To%20Write%20Category%20Identifier/README.md) and [Programmable Block](../../PromptEditor/02-How%20To%20Write%20Prompt%20Content/02-How%20To%20Write%20Programmable%20Block/README.md) |
| **Database Creation Support** | Various Add plugins for importing from file system, clipboard, Civitai, and more |
| **Editing Support** | Various Edit plugins for efficient prompt editing |
| **External App Integration** | Send prompts on demand to external apps like ComfyUI and Stable Diffusion via API server |
| **AI Integration** | Operate via AI assistants like Claude Code through MCP protocol ([AI Auto-Operation](../../Utility/01-Menu/08-AI%20Auto-Operation.md)) |
