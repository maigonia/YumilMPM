# Before Image
**Before Image**

## When to Use
- When you want to browse through multiple images in order
- When you want to go back to a previously viewed image

## What It Does
Displays the previous image.

## How to Use
1. Select "Before Image" from the menu
2. Or click the "◀" button in the toolbar

## Shortcuts
- **←**: Previous image (when the image panel has focus)

## Notes
- Disabled when the first image is displayed
