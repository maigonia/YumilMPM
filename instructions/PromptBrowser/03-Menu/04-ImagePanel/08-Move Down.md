# Move Down
**Move Down**

## When to Use
- When you want to change the display order of images
- When you want to move an image backward

## What It Does
Moves the currently displayed image one position backward in the image list.

## How to Use
1. Display the image you want to move
2. Select "Move Down" from the menu

## Notes
- Disabled when the last image is displayed
- Disabled when there are fewer than 2 images
