# Move Up
**Move Up**

## When to Use
- When you want to change the display order of images
- When you want to move an important image to the front

## What It Does
Moves the currently displayed image one position forward in the image list.

## How to Use
1. Display the image you want to move
2. Select "Move Up" from the menu

## Notes
- Disabled when the first image is displayed
- Disabled when there are fewer than 2 images
