# Remove Image
**Remove Image**

## When to Use
- When you want to remove an image you no longer need from the prompt

## What It Does
Removes the currently displayed image from the prompt. A confirmation dialog is shown.

## How to Use
1. Display the image you want to remove
2. Select "Remove Image" from the menu
3. Click "Remove Image" in the confirmation dialog

## Shortcuts
- **Delete**: Delete (when the image panel has focus)

## Notes
- Only the association with the prompt is removed; the file itself is not deleted
