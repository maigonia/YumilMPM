# Next Image
**Next Image**

## When to Use
- When you want to browse through multiple images in order

## What It Does
Displays the next image.

## How to Use
1. Select "Next Image" from the menu
2. Or click the "▶" button in the toolbar

## Shortcuts
- **→**: Next image (when the image panel has focus)

## Notes
- Disabled when the last image is displayed
