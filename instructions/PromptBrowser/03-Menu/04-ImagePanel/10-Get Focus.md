# Get Focus
**Get Focus**

## When to Use
- When you want to operate images using keyboard shortcuts
- When you want to move focus to the image panel

## What It Does
Moves keyboard focus to the image panel. After focusing, you can operate images with keyboard shortcuts.

## How to Use
1. Select "Get Focus" from the menu

## Notes
Shortcuts available while focused:
- **←**: Previous image
- **→**: Next image
- **Delete**: Delete image
- **Ctrl+V**: Paste image from clipboard
- **Enter**: Full screen
