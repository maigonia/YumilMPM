# Full Screen Image
**Full Screen**

## When to Use
- When you want to view an image at full size
- When you want to check image details

## What It Does
Displays the currently shown image in full screen. Image switching is still available during full screen.

## How to Use
1. Select "Full Screen" from the menu
2. Double-clicking the image also works

## Shortcuts
- **Enter**: Full screen display (when the image panel has focus)

## Notes
- Press Esc to exit full screen
- Use ← → keys to switch images during full screen
