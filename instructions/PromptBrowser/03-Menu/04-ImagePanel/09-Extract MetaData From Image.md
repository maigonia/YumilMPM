# Extract MetaData From Image

**Extract MetaData From Image**

## When to Use

- When you want to check the prompts of images generated by AI image generation tools
- When you want to retrieve information embedded in images

## What It Does

Extracts metadata embedded in image files and displays it in a dialog. You can check prompts and settings of images generated by Stable Diffusion, etc.

## How to Use

1. Display the image whose metadata you want to check
2. Select "Extract MetaData From Image" from the menu
3. The extracted metadata is displayed in a dialog
