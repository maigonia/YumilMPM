# Show/Hide
**Show/Hide**

## When to Use
- When you want to view and edit [Image](../01-Basics/01-Image.md), [Memo](../01-Basics/02-Memo.md), and [Tag](../01-Basics/04-Tag.md) attached to a [Prompt](../../PromptTree/01-Basics/01-Concepts/03-What%20Is%20a%20Prompt.md)
- When you want to close the prompt browser to expand the editor area

## What It Does
Toggles the visibility of the prompt browser panel.

## How to Use
1. Select a prompt in the PromptTree
2. Select "Show/Hide" from the menu

## Notes
- This menu is disabled when no prompt is selected
- The visibility state is preserved on next launch
