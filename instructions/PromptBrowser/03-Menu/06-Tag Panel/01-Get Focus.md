# Get Focus
**Get Focus**

## When to Use
- When you want to add tags using the keyboard
- When you want to move focus to the tag input field

## What It Does
Moves keyboard focus to the tag panel's input field. After focusing, you can immediately start entering tags.

## How to Use
1. Select "Get Focus" from the menu

## Notes
- While entering tags, previously used tags are suggested via autocomplete
- Press Enter to add a tag
- Use arrow keys to select autocomplete suggestions
