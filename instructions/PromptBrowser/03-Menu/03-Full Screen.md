# Full Screen
**Full Screen / Exit Full Screen**

## When to Use
- When you want to expand the prompt browser to fill the entire screen
- When you want to display images or memos larger

## What It Does
Switches the prompt browser to maximized display. In full screen, the entire screen is used.

## How to Use
1. Select "Full Screen" from the menu
2. To exit, select "Exit Full Screen" or click the maximize button in the header

## Notes
- Exiting full screen restores the original size and position
- Full screen works in both docked and floating modes
