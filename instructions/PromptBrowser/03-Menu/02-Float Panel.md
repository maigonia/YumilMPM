# Float Panel
**Float Panel**

## When to Use
- When you want to dock the prompt browser between the prompt tree and prompt editor
- When you want to place the prompt browser as an independent, freely positionable window

## What It Does
Switches the display mode of the prompt browser.

- **Checked (Floating)**: Displayed as an independent window (freely movable and resizable)
- **Unchecked (Docked)**: Fixed between the prompt tree and prompt editor

## How to Use
1. Toggle the "Float Panel" checkbox from the menu

## Notes
- In floating mode, you can drag the title bar to move the window
- In floating mode, you can drag the bottom-right corner to resize
- The display mode and position/size are preserved on next launch
