# Get Focus
**Get Focus**

## When to Use
- When you want to edit memos with the keyboard
- When you want to move focus to the memo panel

## What It Does
Moves keyboard focus to the memo panel's text area. After focusing, you can immediately start editing the memo.

## How to Use
1. Select "Get Focus" from the menu

## Notes
- URLs in memos can be opened in an external browser with Ctrl+Click
