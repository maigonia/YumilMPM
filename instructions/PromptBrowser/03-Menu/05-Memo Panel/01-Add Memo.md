# Add Memo

**Add Memo**

## When to Use

- When you want to add notes about a prompt
- When you want to record generation parameters or reference information

## What It Does

Adds a new memo to the prompt. After adding, focus moves to the new memo.

## How to Use

1. Select "Add Memo" from the menu
2. Or click the "Add" button in the toolbar

## Notes

- Memos can be written in Markdown format and previewed in preview mode
- Multiple memos can be added and navigated between
- Memo content is auto-saved
