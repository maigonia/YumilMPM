# Move Up
**Move Up**

## When to Use
- When you want to change the display order of memos
- When you want to move an important memo to the front

## What It Does
Moves the currently displayed memo one position forward in the memo list.

## How to Use
1. Display the memo you want to move
2. Select "Move Up" from the menu

## Notes
- Disabled when the first memo is displayed
- Disabled when there are fewer than 2 memos
