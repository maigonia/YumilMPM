# Next Memo
**Next Memo**

## When to Use
- When you want to browse through multiple memos in order

## What It Does
Displays the next memo.

## How to Use
1. Select "Next Memo" from the menu
2. Or click the "▶" button in the toolbar

## Notes
- Disabled when the last memo is displayed
- The current memo's content is auto-saved before switching
