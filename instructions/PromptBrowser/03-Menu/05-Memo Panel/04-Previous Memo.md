# Previous Memo
**Previous Memo**

## When to Use
- When you want to browse through multiple memos in order
- When you want to go back to a previously viewed memo

## What It Does
Displays the previous memo.

## How to Use
1. Select "Previous Memo" from the menu
2. Or click the "◀" button in the toolbar

## Notes
- Disabled when the first memo is displayed
- The current memo's content is auto-saved before switching
