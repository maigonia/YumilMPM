# Remove Memo
**Remove Memo**

## When to Use
- When you want to delete a memo you no longer need

## What It Does
Deletes the currently displayed memo. A confirmation dialog is shown.

## How to Use
1. Display the memo you want to delete
2. Select "Remove Memo" from the menu
3. Click "Remove Memo" in the confirmation dialog

## Notes
- Cannot delete if there is only one memo
- After deletion, the previous or next memo is displayed
