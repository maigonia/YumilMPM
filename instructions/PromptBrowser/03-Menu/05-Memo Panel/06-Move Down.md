# Move Down
**Move Down**

## When to Use
- When you want to change the display order of memos
- When you want to move a memo backward

## What It Does
Moves the currently displayed memo one position backward in the memo list.

## How to Use
1. Display the memo you want to move
2. Select "Move Down" from the menu

## Notes
- Disabled when the last memo is displayed
- Disabled when there are fewer than 2 memos
