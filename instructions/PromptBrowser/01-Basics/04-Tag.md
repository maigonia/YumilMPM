# Tag

You can classify and organize prompts by attaching tags to them.

## Adding Tags

1. Enter text in the tag input field
2. Press Enter to add

While entering, previously used tags are suggested via autocomplete. Use arrow keys to select suggestions.

## Removing Tags

Click the × button on a tag to remove it.

Added tags can be managed at GlobalSettings > Tag Manager.
