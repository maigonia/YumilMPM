# Prompt Browser

A panel for viewing and editing [Image](01-Image.md), [Memo](02-Memo.md), and [Tag](04-Tag.md) attached to prompts.

## Panel Layout

Three panels are arranged vertically from top to bottom. Each panel boundary can be resized by dragging.

| Panel | Content |
|-------|---------|
| Image Panel | Add and browse reference images |
| Memo Panel | Edit and preview text memos |
| Tag Panel | Add and remove tags |

## How to Display

1. Select a prompt in the PromptTree
2. Toggle visibility with [Show/Hide](../03-Menu/01-Show Hide.md) from the menu

## Display Modes

- **Docked**: Fixed to the bottom of the screen
- **Floating**: Freely positionable as an independent window

Switch modes using [Float Panel](../03-Menu/02-Float Panel.md).

## Navigation

Use the ◀ ▶ buttons in the header to navigate to the previous/next prompt in the PromptTree.
