# Memo

You can add text memos to prompts. Markdown is supported.

## Basic Operations

| Operation | Method |
|-----------|--------|
| Add | "Add" button in the toolbar |
| Delete | "Remove" button in the toolbar |
| Switch memos | ◀ ▶ buttons |

## Editing and Preview

- **Edit**: Edit text directly
- **Preview**: Render and display Markdown

Content is auto-saved when focus is lost.

## Notes

- Multiple memos can be added to a single prompt
- A memo cannot be deleted if it is the only one
- URLs in preview mode can be opened in a browser by clicking
