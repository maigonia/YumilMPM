# Close Project
**Close Project**

## When to Use
- When you want to finish working on the current project
- When you want to explicitly close the current project before opening another
- When you want to return to the start screen without exiting the app

## What It Does
Closes the currently open project and returns to the start screen. Closing the project clears all data in the app.

## How to Use
1. Select **File > Close Project**
2. A confirmation dialog appears
3. Choose one of the following:
   - **Save and Close**: Save current changes before closing
   - **Close without Saving**: Discard changes and close
4. The start screen is displayed, where a new project can be created or another can be opened

## Notes
- This menu is disabled if no project is open
- Choosing "Close without Saving" will **lose all unsaved changes**
- If "Save and Close" is chosen and [Enable Auto Backup](09-BackUp/02-Quick%20Backup/06-Enable%20Auto%20Backup.md) is on, a backup is also created automatically after saving
- Project settings (generation settings, output settings, etc.) are automatically saved before closing
