# Open Data Folder
**Open Data Folder**

## When to Use
- When you want to directly check the project files
- When you want to back up the data
- When you want to edit files with an external tool

## What It Does
Opens the current project folder in Explorer (Finder).

## How to Use
Select **File > Open Data Folder**

## Notes
- This menu is unavailable if no project is open
- The opened folder contains:
  - `/Data` - JSON files for categories, prompts, settings, etc.
  - `/Prompts` - Output destination for generated prompts
  - `project_state.json` - Project marker file

## Tips
- If you edit files directly in the folder, use **Reload Current Project** to reflect the changes in the app
- It is recommended to create a backup before modifying important data
