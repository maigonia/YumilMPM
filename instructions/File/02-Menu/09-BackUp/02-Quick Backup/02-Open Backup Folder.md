# Open Backup Folder
**Open Backup Folder**

## When to Use
- When you want to check backup files
- When you want to manage backups manually
- When you want to copy backup files to another location
- When you want to check storage usage

## What It Does
Opens the folder where backups are saved in Explorer (Finder).

## How to Use
Select **File > BackUp > Quick Backup > Open Backup Folder**

## Notes
- If the backup folder doesn't exist, it is automatically created before opening
- By default, the `/BackUp` folder inside the project folder is opened
- If a custom backup folder is configured, the configured folder is opened
- Backup files are saved in the format `backup_YYYYMMDD_HHmmss.zip`

## Tips
- Useful when copying backup files to external storage
- You can check when a backup was created from the date in the file name
