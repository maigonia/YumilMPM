# Backup Folder Settings
**Backup Folder Settings**

## When to Use
- When you want to check the current backup status
- When you want to see the number of backups
- When you want to check how much storage backups are using

## What It Does
Displays backup folder information:
- Number of backup files
- Total file size

## How to Use
Select **File > BackUp > Quick Backup > Backup Folder Settings...**

## Notes
- The displayed size is the total of quick backup ZIP files
- Full backup sizes are not included
- If a custom backup folder is configured, that folder's information is displayed

## Tips
- If backups have accumulated too much, clean up with [Delete Older Than 30 Days](04-Delete%20Older%20Than%2030%20Days.md) or [Keep Only Last 30 Backups](05-Keep%20Only%20Last%2030%20Backups.md)
- It is recommended to periodically check backup status and clean up as needed
