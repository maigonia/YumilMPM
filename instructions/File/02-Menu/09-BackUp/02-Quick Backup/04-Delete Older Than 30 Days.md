# Delete Older Than 30 Days
**Delete Backups Older Than 30 Days**

## When to Use
- When you want to clean up old backups
- When you want to save storage space
- When you want to clean up the backup folder

## What It Does
Batch-deletes backup files that are more than 30 days old from their creation date.

## How to Use
1. Select **File > BackUp > Quick Backup > Delete Older Than 30 Days**
2. A confirmation dialog appears
3. Click **Delete** to execute

## Notes
- A confirmation dialog is always displayed before deletion
- The number of deleted files is shown in the status bar
- Only ZIP files in the backup folder are targeted
- **This operation cannot be undone**

## Tips
- Combine with [Keep Only Last 30 Backups](05-Keep%20Only%20Last%2030%20Backups.md) for more effective backup management
- Regular cleanup helps make good use of storage
