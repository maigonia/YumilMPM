# Open Project
**Open Project...**

## When to Use
- When you want to resume a previously created project
- When you want to switch to a different project
- When you want to open a project from another location

## What It Does
Opens an existing project folder and loads the saved data. All data including [Category](../../AdvancedPanel/01-Basics/01-Category%20Template%20Tab.md), [Prompt](../../PromptTree/01-Basics/01-Concepts/03-What%20Is%20a%20Prompt.md), [Tag](../../PromptBrowser/01-Basics/04-Tag.md), [Memo](../../PromptBrowser/01-Basics/02-Memo.md), and [Image](../../PromptBrowser/01-Basics/01-Image.md) is restored.

## How to Use
1. Select **File > Open Project...** (or shortcut `Ctrl+O`)
2. A folder selection dialog appears
3. Click the "Browse..." button
4. Select an **existing project folder**
5. Click the "Open Project" button
6. The project is loaded and work can resume from the previous state

## Notes
- The selected folder **must contain existing project data**
- If a folder without a project is selected, a "Project not found" warning is displayed
- When opening a project, the following settings are also restored:
  - Font settings
  - Editor keybinding settings
  - Window position and size
- If a project is currently open, [AutoSave on Application Close](08-AutoSave%20on%20Application%20Close.md) / [Enable Auto Backup](09-BackUp/02-Quick%20Backup/06-Enable%20Auto%20Backup.md) (when enabled) silently save and back up the previous project before switching (no dialog)
