# Save Project
**Save Project**

## When to Use
- When you want to explicitly save your work
- When you want to make sure changes are saved after making important edits

## What It Does
Saves all data of the current project to disk. Main data includes:
- [Category](../../AdvancedPanel/01-Basics/01-Category Template Tab.md) tree structure
- [Prompt](../../PromptTree/01-Basics/01-Concepts/03-What Is a Prompt.md) (chunk) data for each category
- Check state for each prompt
- Per-category settings

Tags, favorite queries, window state, and more are also saved.

## How to Use
Select **File > Save Project** (or shortcut `Ctrl+S`)

## Notes
- If the editor has unsaved content, it is automatically applied before saving
- The check states in the prompt tree are also saved
- An error message is displayed if saving fails

## Tips
- Normally, this app saves data automatically, so manual saving is rarely needed
- It's reassuring to explicitly save at important milestones in your work
