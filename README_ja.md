## 機能

| 機能 | 説明 |
|---|---|
| **ツリー管理** | プロンプトをフォルダ・ファイル形式の階層構造で整理。 |
| **カテゴリ分類** | 「品質」「キャラクター」「背景」など目的別にカテゴリを作成。 |
| **チェック選択** | チェックボックスで使いたいプロンプトを選択。 |
| **動的文字列置換** | Category IdentifierやProgrammable Blockによるカテゴリ間参照・動的プロンプト構築。 |
| **データベース作成支援** | ファイルシステム・クリップボード等からの多彩なAdd系プラグイン。 |
| **編集支援** | プロンプトの効率的な編集をサポートする各種Edit系プラグイン。 |
| **外部アプリ連携** | APIサーバー経由でComfyUI・Stable Diffusion等の外部アプリへオンデマンドでプロンプトを送信。 |
| **キュー管理** | 複数のプロンプト生成タスクをキューに登録し、順番に処理。 |
| **AI連携** | MCPプロトコルでClaude Code等のAIアシスタントから操作可能。 |

### 動作環境

| | 要件 |
|---|---|
| **OS** | Windows 10/11 |
| **アーキテクチャ** | x64 |

### Linux / macOS

実験的・サポート対象外のビルドを [GitHub Releases](https://github.com/maigonia/YumilMPM/releases/latest) で公開しています。これらはコンパイルは通っていますが、作者による動作確認・署名は行えておらず、Microsoft Store 版のような正式サポートはご提供できません。

- **Linux**: `.deb`（Debian/Ubuntu）/ `.AppImage`（汎用）、x86_64
- **macOS**: `.dmg`、Apple Silicon 専用（M1/M2/M3/M4）

## 拡張機能

Yumil MPMを画像生成ワークフローに統合。

| 拡張機能 | 説明 |
|---|---|
| **[ComfyUI 拡張](https://github.com/maigonia/comfyui-yumil-mpm)** | ExternalPromptRequesterノードとMPM Image Parserノードで、ComfyUIとシームレスに連携。 |
| **[Stable Diffusion WebUI 拡張](https://github.com/maigonia/sd-webui-yumil-mpm)** | Stable Diffusion WebUIインターフェースからプロンプトを直接送受信。 |
