# Support

Yumil MPM is developed and maintained by a single individual. Replies may take some time; thank you for your patience.

Yumil MPM は個人開発のアプリです。返答までにお時間をいただく場合があります。ご了承ください。

---

## Where to ask / どこで聞くか

| Type | Where |
|---|---|
| 🐛 Bug in the main app / 本体のバグ | [Issues on this repo](https://github.com/maigonia/YumilMPM/issues) |
| ✨ Feature request / 機能要望 | [Issues on this repo](https://github.com/maigonia/YumilMPM/issues) |
| 💬 Usage question / 使い方の質問 | [Discussions](https://github.com/maigonia/YumilMPM/discussions) |
| 🧩 ComfyUI integration issue | [comfyui-yumil-mpm Issues](https://github.com/maigonia/comfyui-yumil-mpm/issues) |
| 🎨 SD WebUI integration issue | [sd-webui-yumil-mpm Issues](https://github.com/maigonia/sd-webui-yumil-mpm/issues) |
| 📧 Privacy inquiry / プライバシー問い合わせ | yumil.mpm@gmail.com |
| 🔒 Security vulnerability / セキュリティ脆弱性 | See [SECURITY.md](SECURITY.md) — please report privately, not via public Issues. |

---

## Languages / 使用言語

Issues, Pull Requests, and Discussions can be written in **English or Japanese**.
Issue・Pull Request・Discussion は **英語または日本語**で記入できます。
Replies will match the language of the original post.
返答は最初に書かれた言語に合わせます。

---

## Before opening an Issue / Issue を立てる前に

- Please **search existing Issues and Discussions** first to see if the topic is already covered.
- 既存の Issue / Discussion を **検索**してから新規投稿してください。
- For bug reports, **screenshots of error dialogs** and detailed reproduction steps are very helpful.
- バグ報告時は **エラーダイアログのスクリーンショット** と詳細な再現手順をご提供いただけると助かります。

---

## What is out of scope / 対応しないもの

- Closed-source modifications: the main app source is in a private repository; we do not accept code-modification PRs here.
- メイン本体のコード変更 PR: 本体ソースは Private リポにあるため、ここでは受け付けません。
- Custom plugin development support: see the in-app instructions for the plugin SDK.
- カスタムプラグイン開発のサポート: アプリ内のインストラクションを参照してください。

For extensions licensed under MIT (`comfyui-yumil-mpm`, `sd-webui-yumil-mpm`), code contributions are welcome on those repositories.
MIT ライセンスの拡張機能（`comfyui-yumil-mpm`、`sd-webui-yumil-mpm`）へのコード貢献は各リポでお待ちしています。
